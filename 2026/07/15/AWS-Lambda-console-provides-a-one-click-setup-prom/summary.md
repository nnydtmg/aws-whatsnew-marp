# AWS Lambda コンソールのコーディングエージェント向けワンクリックセットアップ

AWS Lambda console provides a one-click setup prompt for coding agents

**カテゴリ:** What's New
**公開日:** 2026-07-14T20:00:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/07/aws-lambda-prompt-coding-agents/)

このページでは、AWS What's Newで発表された「AWS Lambda console provides a one-click setup prompt for coding agents」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

AWS Lambda コンソールにワンクリックセットアッププロンプトが追加され、開発者はコーディングエージェントを AWS Serverless スキルと Serverless MCP サーバーで簡単に構成できるようになりました。このアップデートは、Lambda 開発を効率化したいすべての開発者、特に複数のコーディングエージェントを使用している開発者に有益です。

## このアップデートで何が変わったか

### 新機能について

- AWS Lambda コンソールにコーディングエージェント用のワンクリックセットアッププロンプトが提供されるようになりました。
- このセットアップにより、AWS Serverless スキルと Serverless Model Context Protocol (MCP) サーバーが自動的に構成されます。
- サーバーレスのベストプラクティスが初期段階から組み込まれています。
- Lambda コンソールのあらゆる段階（入門、機能探索、初回関数作成）でこのセットアップが利用可能です。

### このアップデートが適している対象者について

- Lambda 関数の構築、テスト、デプロイにコーディングエージェントを使用している開発者に適しています。
- 複数のドキュメントページを参照することなく、エージェントを簡単にセットアップしたい開発者に適しています。
- Claude Code、Kiro、Cursor、GitHub Copilot、Codex、Devin Desktop、OpenCode などの様々なコーディングエージェントを利用している開発者に適しています。
- AWS 認証がローカルに構成されていない開発者もサポートされています。

# Amazon OpenSearch Service が Agent Toolkit for AWS に統合、AI エージェントから自然言語で管理可能に

Amazon OpenSearch Service now supports the Agent Toolkit for AWS with a curated skill

## 要約

Amazon OpenSearch ServiceがAgent Toolkit for AWSに統合され、AIコーディングエージェントから自然言語でOpenSearchドメインの管理、検索、ログ分析、トレース分析が可能になりました。既存インフラへの変更は不要で、追加費用なしにすべてのAWSリージョンで利用できます。

## このアップデートで何が変わったか

**新機能について:**
- Amazon OpenSearch ServiceがAgent Toolkit for AWSに対応し、Claude Code、Kiro、CursorなどのAIコーディングエージェントから直接OpenSearch ServiceドメインとOpenSearch Serverlessコレクションを構築、管理、クエリできるようになりました。

- 既存のインフラストラクチャへの変更は不要で、追加費用なしで利用できます。

## 詳細

**新機能について:**
- Amazon OpenSearch ServiceがAgent Toolkit for AWSに対応し、Claude Code、Kiro、CursorなどのAIコーディングエージェントから直接OpenSearch ServiceドメインとOpenSearch Serverlessコレクションを構築、管理、クエリできるようになりました。

- AWS MCP（Model Context Protocol）サーバーとamazon-opensearch-serviceスキルの組み合わせにより、自然言語リクエストが適切な機能に自動的にルーティングされます。

- 5つの主要領域に対応しており、マイグレーション、オペレーション、検索（ベクトル、セマンティック、ハイブリッド、RAG検索）、ログ分析、トレース分析が含まれています。

- 既存のインフラストラクチャへの変更は不要で、追加費用なしで利用できます。

- すべてのAWSリージョンでOpenSearch ServiceおよびOpenSearch Serverlessが提供されている場所で利用可能です。

**この更新が適している対象:**
- OpenSearch Serviceを使用している開発者およびアーキテクト

- AIコーディングエージェントを活用して開発効率を向上させたい組織

- 自然言語でOpenSearchドメインやコレクションを管理したいユーザー

- ベクトル検索、ログ分析、トレース分析などの高度な機能を必要とするチーム

- セルフマネージドOpenSearchからOpenSearch Serviceへの移行を検討している企業

# AWS Security Hub が Microsoft Azure のセキュリティ管理に対応

AWS Security Hub extends unified security management to Microsoft Azure

**カテゴリ:** What's New
**公開日:** 2026-06-30T12:21:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-security-hub-supports-monitoring-microsoft-azure/)

このページでは、AWS What's Newで発表された「AWS Security Hub extends unified security management to Microsoft Azure」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

AWS Security Hubが Microsoft Azureのリソース監視に対応し、AWSとAzureの両環境のセキュリティ検出結果を統一されたコンソールから管理できるようになりました。このアップデートは、マルチクラウド環境でセキュリティ運用を効率化したいお客様に適しており、30日間の無料トライアルが提供されます。

## このアップデートで何が変わったか

- Azure統合作成後、30日間の無料トライアルが利用可能です。

- 複数のAWSリージョンで利用可能ですが、中東（UAE）、中東（バーレーン）、アジア太平洋（台北）、アジア太平洋（ニュージーランド）では利用できません。

## 対象ユーザー

- CIS Benchmarks for Microsoft Azure Foundationsなどのセキュリティ標準に対するポスチャチェック、統一されたリソースインベントリ、リスク分析、およびEventBridge統合による自動対応が提供されます。

## 詳細

- AWS Security Hubが、Microsoft Azureのリソース監視に対応するようになりました。

- Azureの仮想マシン、コンテナレジストリ、Function Apps、および認証情報を自動的に検出し、設定ミス、インターネット露出、ソフトウェア脆弱性について評価します。

- AWS環境とAzure環境のセキュリティ検出結果が同一の優先順位付きビューで表示され、統一されたフォーマットと自動化ワークフローで対応できます。

- CIS Benchmarks for Microsoft Azure Foundationsなどのセキュリティ標準に対するポスチャチェック、統一されたリソースインベントリ、リスク分析、およびEventBridge統合による自動対応が提供されます。

- Azure統合作成後、30日間の無料トライアルが利用可能です。

- トライアル終了後は、AWS リソースと同等の価格でAzureリソース監視の料金が発生します。

- 複数のAWSリージョンで利用可能ですが、中東（UAE）、中東（バーレーン）、アジア太平洋（台北）、アジア太平洋（ニュージーランド）では利用できません。

- このアップデートは、AWS と Azure の両方で運用負荷を軽減し、統一されたセキュリティ管理を実現したい組織に適しています。

- マルチクラウド環境を運用するセキュリティチームが、複数のツールを切り替える手間を削減できます。

# AWS Security Hub が露出検出の影響分析機能をサポート

AWS Security Hub adds impact analysis for exposure findings

**What's New** | 2026-07-06T21:45:00

---

## 概要

- AWS Security Hubは、露出検出に対する影響分析機能を追加し、攻撃者が悪用可能なリソースの下流への到達範囲を可視化することで、セキュリティチームがより効果的にリスクを評価・優先順位付けできるようになりました。
- この機能は、IAMの権限昇格パスを分析し、重大度スコアリングに反映させることで、組織全体のセキュリティ態勢の向上を支援いたします。

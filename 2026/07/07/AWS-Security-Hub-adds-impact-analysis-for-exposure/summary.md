# AWS Security Hub が露出検出の影響分析機能をサポート

AWS Security Hub adds impact analysis for exposure findings

## 要約

AWS Security Hubは、露出検出に対する影響分析機能を追加し、攻撃者が悪用可能なリソースの下流への到達範囲を可視化することで、セキュリティチームがより効果的にリスクを評価・優先順位付けできるようになりました。この機能は、IAMの権限昇格パスを分析し、重大度スコアリングに反映させることで、組織全体のセキュリティ態勢の向上を支援いたします。

## このアップデートで何が変わったか

- AWS Security Hubは、露出検出（exposure findings）に対して影響分析機能を追加いたしました。

- このアップデートは、セキュリティチームと組織のリスク管理に責任を持つ専門家に特に有益です。

## 詳細

- AWS Security Hubは、露出検出（exposure findings）に対して影響分析機能を追加いたしました。

- この機能により、セキュリティチームは、露出が悪用された場合に攻撃者が到達可能な範囲を完全に理解することができるようになりました。

- 影響分析は、初期的に露出したリソースを超えて、侵害される可能性のある下流のリソースをマッピングすることで、組織的なリスクに対するより深い可視性を提供いたします。

- Security Hubは、露出したリソースに関連するIAMプリンシパルの有効なアクセス権限を分析し、アカウント内の他のリソースへの権限昇格パスを特定いたします。

- 影響範囲は、潜在的な攻撃パスグラフに表示され、新しい「Impact Assessment」タブには、攻撃者が辿る可能性のあるリソースチェーンと各ステップでの特定のアクセス権限が優先順位付けされて表示されます。

- Security Hubは、露出検出の重大度スコアリングに影響範囲を組み込み、影響範囲が特定されたり変更されたりする際に既存の露出を調整いたします。

- これにより、下流への到達範囲が大きい露出が適切に優先順位付けされるようになりました。

- このアップデートは、セキュリティチームと組織のリスク管理に責任を持つ専門家に特に有益です。

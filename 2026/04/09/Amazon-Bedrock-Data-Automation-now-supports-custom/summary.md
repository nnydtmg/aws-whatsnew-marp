# Amazon Bedrock Data Automation now supports custom vocabulary

**カテゴリ:** What's New  
**公開日:** 2026-04-03T21:29:00  
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/bedrock-data-automation-vocabulary/](https://aws.amazon.com/about-aws/whats-new/2026/04/bedrock-data-automation-vocabulary/)

---

## 要約

Amazon Bedrock Data Automationが新たに**カスタム語彙機能**をサポート開始しました。この機能により、ドメイン固有の専門用語の音声認識と文字起こし精度が向上します。医療、法律、金融サービス、メディア、コンタクトセンター分析などの業界に特に有用です。11言語に対応し、追加料金なしで利用できます。

---

## 詳細

### 新機能の概要

- **カスタム語彙機能の実装**: Amazon Bedrock Data Automationにカスタム語彙機能が新たに追加されました
- **精度向上**: ドメイン固有の用語に対する音声認識と文字起こしの精度が大幅に向上します
- **対応範囲**: ブランド名、頭字語、医学用語、技術用語などの専門用語をリストとして提供できます
- **表示形式カスタマイズ**: 認識された用語の表示形式をカスタマイズでき、例えば「electrocardiogram」を「ECG」と表示することが可能です

### 言語対応

**11言語サポート:**
- 英語、スペイン語、フランス語、ドイツ語、イタリア語、ポルトガル語
- 日本語、韓国語
- 中国語（簡体字、繁体字、広東語）

### コスト

- **追加料金なし**: この機能は追加料金なしで利用できます
- **既存利用者対象**: Amazon Bedrock Data Automation の既存ユーザーが即座に活用可能

### 適用業界

- **医療業界**: 医師と患者の会話処理
- **法律業界**: 法廷会議や契約書の処理
- **金融サービス**: 金融取引の記録保持
- **メディア業界**: 多言語コンテンツの文字起こし
- **コンタクトセンター**: コンタクトセンター分析プロバイダーの品質向上

### 利用可能リージョン

- 複数のAWSリージョンで利用可能です

### 対象ユーザー

- 医療業界で医師と患者の会話を処理する企業
- コンタクトセンター分析プロバイダーで複数の業界に対応する企業
- 専門用語の正確な認識が必要な金融サービス企業
- 法律業界で専門用語の精密な文字起こしが必要な組織
- メディア業界で多言語コンテンツを処理する企業

## 関連技術・サービス

- **Amazon Bedrock**: AWS の生成AI サービス
- **Data Automation Library**: カスタム語彙を管理するライブラリ機能
- **Bedrock Data Automation API**: プログラマティックアクセス

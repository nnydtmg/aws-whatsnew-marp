# Amazon Quick Automate announces shared file storage for automations

**カテゴリ:** What's New
**公開日:** 2026-04-21
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-quick-automate-shared-file-storage/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-quick-automate-shared-file-storage/)

---

## 要約

Amazon Quick Automateの新しい共有ファイルストレージ機能により、外部ストレージソリューションなしでオートメーション内でファイルを一元管理できるようになりました。この機能は、複数のオートメーション間でファイルを共有する必要があるオートメーションビルダーおよびビジネスユーザーに特に有益です。

## 詳細

Amazon Quick Automateが新しい共有ファイルストレージ機能を発表いたしました。この機能により、外部ストレージソリューションやコネクタを必要とせず、オートメーション内でシームレスにファイルを管理できるようになりました。ドラッグ&ドロップインターフェースを使用して、ファイルを直接アップロードし、同じオートメーショングループ内のすべてのオートメーション間で即座にアクセスできます。従来は、Amazon S3、OneDrive、SharePointなどの外部サービスへのアップロードとコネクタの設定が必要でした。連絡先リストや価格表などの静的ルックアップファイル、ビジネスユーザーが管理する構成ファイル、複数のオートメーション間でのデータフロー、日次または週次レポート共有などのユースケースに対応しています。この機能は、Quick Automateが有効になっているすべてのAWSリージョンで利用可能です。

# Amazon CloudWatch pipelines now supports configuration of processors via AI

**カテゴリ:** What's New
**公開日:** 2026-04-21T16:30:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-cloudwatch-pipelines-ai-configuration/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-cloudwatch-pipelines-ai-configuration/)

---

## 要約

Amazon CloudWatch パイプラインに、生成AIを活用した自然言語によるプロセッサ設定機能が追加されました。ユーザーはプレーンテキストで処理内容を記述するだけで、パイプライン設定が自動生成され、セットアップ時間の短縮と複雑なログ処理の簡素化が実現されます。

## 詳細

- Amazon CloudWatch パイプラインが、生成AIを活用した自然言語によるプロセッサ設定機能をサポートするようになりました。
- ユーザーは複雑なログ形式の解析と変換を、プレーンテキストの説明で指定できるようになりました。
- CloudWatch コンソールの処理ステップでAI支援オプションを有効にし、自然言語で変換内容を記述することで、パイプライン設定が自動生成されます。
- サンプルログイベントとともに処理結果が表示されるため、デプロイ前に出力内容を検証できます。
- この機能により、セットアップ時間が短縮され、プロセッサ設定の深い知識がなくてもパイプラインを正しく実行できるようになります。
- AI支援プロセッサ設定は、CloudWatch パイプラインが利用可能なすべてのAWSリージョンで追加費用なしで提供されます。
- 標準的なCloudWatch Logs の取り込みおよびストレージ料金は引き続き適用されます。

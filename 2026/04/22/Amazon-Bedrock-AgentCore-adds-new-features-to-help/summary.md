# Amazon Bedrock AgentCore adds new features to help developers build agents faster

**カテゴリ:** What's New
**公開日:** 2026-04-22T16:00:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/agentcore-new-features-to-build-agents-faster/](https://aws.amazon.com/about-aws/whats-new/2026/04/agentcore-new-features-to-build-agents-faster/)

---

## 要約

Amazon Bedrock AgentCoreは、マネージドハーネス、AgentCore CLI、およびコーディングアシスタント向けスキルを新たに導入することで、開発者がエージェントプロトタイプをより迅速に構築し、完全なエージェントライフサイクルを単一プラットフォームから管理できるようにいたしました。本アップデートは、プロトタイプから本番環境への移行を効率化し、ガバナンスと監査可能性を備えたエージェント開発を実現したい開発チームに特に有益です。

## 詳細

**新機能について:**
- Amazon Bedrock AgentCoreが、開発者がエージェントプロトタイプをより迅速に構築できるようにするための新機能を導入いたしました。
- マネージドハーネス（プレビュー）により、モデル、システムプロンプト、ツールを指定するだけで、オーケストレーションコードなしにエージェントを実行できるようになりました。
- ハーネスは推論、ツール選択、アクション実行、レスポンスストリーミングの完全なエージェントループを管理いたします。
- AgentCore CLIにより、インフラストラクチャアズコードのガバナンスと監査可能性を備えたデプロイメントが可能になりました。
- ファイルシステム永続化（プレビュー）により、エージェントがタスクの途中で中断し、中断した箇所から正確に再開できるようになりました。
- AgentCore スキルにより、コーディングアシスタント向けの事前構築されたスキルが提供されます。

**このアップデートが適している対象者:**
- エージェント開発を迅速に開始したい開発者にとって有益です。
- プロトタイプから本番環境への移行を効率的に行いたい開発チームに適しています。
- コーディングアシスタントの統合を検討しているエンジニアに役立ちます。
- インフラストラクチャアズコードを用いたガバナンスを重視する組織に適しています。

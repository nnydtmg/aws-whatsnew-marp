# Amazon S3 Express One Zone now supports S3 Inventory

**カテゴリ:** What's New
**公開日:** 2026-04-20T13:00:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/s3-express-one-zone-supports-s3-inventory/](https://aws.amazon.com/about-aws/whats-new/2026/04/s3-express-one-zone-supports-s3-inventory/)

---

## 要約

Amazon S3 Express One ZoneがS3 Inventoryをサポート開始し、日次または週次のスケジュール済みレポートでオブジェクトのメタデータと暗号化ステータスを管理できるようになりました。このアップデートは、レイテンシーに敏感なアプリケーションを運用し、ビジネスコンプライアンスと規制要件を満たす必要があるユーザーに特に有益です。

## 詳細

- Amazon S3 Express One Zoneが、レイテンシーに敏感なアプリケーション向けの高性能ストレージクラスとして、S3 Inventoryのサポートを開始いたしました。

- S3 Inventoryは、S3の同期的なList APIに代わるスケジュール済みの代替手段を提供いたします。

- 日次または週次ベースでレポートを生成し、S3ディレクトリバケット内またはプレフィックス配下に保存されたオブジェクトとそれぞれのメタデータおよび暗号化ステータスをリスト化することができます。

- AWS CLI、AWS SDK、またはS3 APIを使用して、日次または週次のインベントリレポートを設定できます。

- 出力ファイル形式としてCSV、ORC、またはParquetから選択可能です。

- オブジェクト名、サイズ、最終更新日、ストレージクラス、マルチパートアップロードフラグ、暗号化ステータスなどのメタデータを指定できます。

- このアップデートは、S3 Express One Zoneが利用可能なすべてのAWSリージョンで提供されます。

- ビジネスワークフローとビッグデータジョブの簡素化と高速化が可能になります。

- コンプライアンスおよび規制要件を満たすため、オブジェクトの暗号化ステータスを検証できます。

- このアップデートは、レイテンシーに敏感で、スケジュール済みのインベントリレポートが必要なユーザーに適しています。

- ビッグデータ処理やコンプライアンス検証が必要な組織に有益です。

- 暗号化ステータスの確認とメタデータ管理を効率化したいユーザーに最適です。

# Amazon Bedrock AgentCore Browser
OS レベルインタラクション機能が新登場

**発表日: 2026年4月8日**

---

## 概要

### Amazon Bedrock AgentCore Browser の進化

- **OS レベルのインタラクション機能を新たに追加**
  - マウス・キーボード操作に対応
  - デスクトップ全体のスクリーンショット取得

- **CDP では対応できない複雑なブラウザワークフローの自動化が可能に**
  - プリントダイアログ、ネイティブシステムアラート、右クリックメニューなど
  - より複雑なユーザーインタラクションを自動実行

- **14 の AWS リージョンでデフォルト利用可能**
  - すぐに本番環境で活用可能

---

## 前提・背景

### 従来の課題

- **Chrome DevTools Protocol（CDP）の限界**
  - 標準的なブラウザ自動化では対応できない複雑な操作
  - ネイティブシステムダイアログへのアクセス不可
  - OS レベルのキーボードショートカットが使用不可

### AI エージェント技術の進化

- **Amazon Bedrock AgentCore の拡張**
  - Stateful MCP Server サポート（2026年3月）
  - ビジョンベースの AI エージェント開発への対応
  - 複数ステップの自動化タスクへの対応能力向上

---

## 主な機能

### 1. マウス操作

- **クリック**: 指定された座標への正確なクリック
- **移動**: マウスカーソルの移動
- **ドラッグ**: 要素のドラッグ・ドロップ操作
- **スクロール**: ページやウィンドウのスクロール

### 2. キーボード操作

- **テキスト入力**: 通常のテキスト入力
- **キー押下**: 個別のキーやキーの組み合わせ
- **ショートカット**: Ctrl+A、Ctrl+C、Ctrl+P などの実行

---

## 主な機能（続き）

### 3. ビジュアル認識

- **デスクトップスクリーンショット**: 画面全体をキャプチャ
- **ビジョンベース判定**: スクリーンショットから現在の状態を認識
- **複雑な UI 操作の自動化**: 視覚情報に基づいた正確な操作

### 4. 対応するユースケース

- **プリントダイアログ**: 印刷設定と実行
- **ネイティブシステムアラート**: OS が発行するダイアログへの対応
- **右クリックメニュー**: コンテキストメニュー操作
- **複雑なフォーム入力**: マルチステップの入力処理

---

## 対象ユーザーと利用シーン

### 適用対象

- **AI エージェント開発者**
  - より複雑なエージェント処理の実装
  - ビジョンベース判定による自動化

- **テスト自動化エンジニア**
  - 従来のテストツールでは対応できない複雑なシナリオ
  - マルチステップの統合テスト

- **LLM 駆動型ツール構築組織**
  - Web インタラクションツール
  - RPA（ロボティック・プロセス・オートメーション）実装

### 主なユースケース

- **自動テスト**: UI テストの自動化
- **ドキュメント管理ワークフロー**: フォーム入力と処理の自動化
- **ビジョンベース AI エージェント**: 画面認識に基づくタスク実行

## 利用開始方法

### 前提条件

- Amazon Bedrock AgentCore への アクセス権
- AWS CLI または SDK を使用できる環境
- 対応リージョンでのリソース構築

### 対応リージョン（14 リージョン）

- US East（N. Virginia）、US West（Oregon）
- EU（Ireland）、EU（Frankfurt）
- Asia Pacific（Tokyo）、Asia Pacific（Singapore）
- その他主要リージョン

### 実装パターン

```
1. Amazon Bedrock AgentCore の初期化
2. Browser エージェントの作成
3. OS レベルアクション API の呼び出し
4. ビジュアルフィードバックに基づく処理継続
```

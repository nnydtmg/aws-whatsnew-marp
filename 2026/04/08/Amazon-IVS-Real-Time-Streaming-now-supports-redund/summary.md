# Amazon IVS Real-Time Streaming now supports redundant ingest

**カテゴリ:** What's New
**公開日:** 2026-04-08T19:25:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ivs-real-time-streaming-redundant-ingest/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ivs-real-time-streaming-redundant-ingest/)

---

## 要約

Amazon IVS Real-Time Streamingの冗長インジェスト機能により、2つのエンコーダーからの同時ストリーミングと自動フェイルオーバーが可能になり、ソースエンコーダー障害やネットワーク問題に対する保護が実現されました。この機能は、ライブイベントや24時間365日のストリーミングなど、中断のない配信が必須となるシナリオに最適です。

## 詳細

- Amazon IVS Real-Time Streamingが冗長インジェスト機能をサポートするようになりました。

- ソースエンコーダーの障害とファーストマイルネットワークの問題からライブストリームを保護するのに役立ちます。

- 2つのエンコーダーから同時に単一のステージにストリーミングでき、自動フェイルオーバー機能が備わっています。

- 視聴者への中断のない配信を確保することができます。

- ライブイベント、24時間365日のライブストリーム、または中断のない配信が重要なシナリオに最適です。

- 予期しない中断時に視聴者のエンゲージメントを維持し、継続的な24時間365日のストリーミングを実現できます。

- この機能は、低遅延またはリアルタイムビデオを世界中の視聴者に提供するマネージドライブストリーミングソリューションの一部です。

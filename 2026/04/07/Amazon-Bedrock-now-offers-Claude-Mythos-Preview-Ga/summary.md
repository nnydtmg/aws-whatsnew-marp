# Amazon Bedrock now offers Claude Mythos Preview (Gated Research Preview)

**カテゴリ:** What's New
**公開日:** 2026-04-07
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-bedrock-claude-mythos/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-bedrock-claude-mythos/)

---

## 要約

Amazon Bedrockは、サイバーセキュリティとソフトウェアコーディングにおいて最先端の能力を持つClaude Mythos Previewをゲート付きリサーチプレビューで提供開始いたしました。このアップデートは、セキュリティチームが脆弱性を迅速に特定・修正し、インターネット関連企業とオープンソースメンテナーを優先して防御的なサイバーセキュリティ作業を加速させるために最適です。

## 詳細

### Claude Mythos Preview について

- Amazon Bedrockは、Claude Mythos Previewをゲート付きリサーチプレビューとして提供開始いたしました。
- Claude Mythos Previewは、Anthropicが開発した最先端のAIモデルであり、サイバーセキュリティ、ソフトウェアコーディング、複雑な推論タスクにおいて最高水準の能力を備えております。
- このモデルは、ソフトウェア内の高度なセキュリティ脆弱性を特定し、その悪用可能性を実証することができます。
- 大規模なコードベースを理解し、以前のAIモデルよりも少ない手動ガイダンスで実行可能な知見を提供いたします。

### セキュリティチームへのメリット

- セキュリティチームは、防御的なサイバーセキュリティ作業を加速し、世界中の重要なソフトウェアのセキュリティ脆弱性を発見・修正することが可能になります。
- 高度な脆弱性の自動検出により、人的リソースの効率化が実現します。
- 大規模コードベースの脆弱性管理が従来より大幅に効率化されます。

### リリース方針

- AnthropicとAWSは、インターネット関連企業とオープンソースメンテナーを優先する慎重なリリースアプローチを採用しております。
- このアプローチにより、防御者が十分に準備できる環境を整備し、業界全体がセキュリティ向上の恩恵を受ける機会を提供いたします。

### 提供方法

- Claude Mythos Previewは、米国東部（バージニア北部）リージョンのAmazon Bedrockを通じてゲート付きプレビューで利用可能です。
- アクセスは初期の許可リストに登録された組織に限定されており、許可リストに登録された組織のAWSアカウントチームが直接連絡いたします。
- Anthropic と AWS が合計で最大 $100 million の使用クレジットを提供予定です。

### 業界への影響

- このモデルは、セキュリティチームがコードベースを強化し、業界全体が恩恵を受けるよう知見を共有する機会を提供いたします。
- Claude Mythos は単なるバージョンアップではなく、AIモデルの新たなクラスを代表する存在です。
- サイバーセキュリティにおける「watershed moment（転換点）」として位置付けられています。

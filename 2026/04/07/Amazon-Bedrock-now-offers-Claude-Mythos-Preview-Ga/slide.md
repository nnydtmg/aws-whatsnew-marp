# Claude Mythos Preview が Amazon Bedrock で利用開始

サイバーセキュリティとコーディングにおける最先端のAIモデル

**発表日: 2026年4月7日**

---

## 概要

### Claude Mythos Preview の3つの主要特徴

- **サイバーセキュリティ対応**: ソフトウェア内の高度なセキュリティ脆弱性を特定し、悪用可能性を実証
- **Advanced Coding**: 大規模なコードベースを理解し、少ない手動ガイダンスで実行可能な知見を提供
- **複雑な推論**: 複数ステップの推論タスクを自律的に実行し、業界最高水準の性能を実現

### 提供方式

- **ゲート付きリサーチプレビュー**: 初期段階での慎重なリリース
- **利用可能リージョン**: 米国東部（バージニア北部）
- **アクセス制限**: 許可リストに登録された組織のみ

---

## 前提・背景

### サイバーセキュリティの課題

- 高度なセキュリティ脆弱性の発見と修正に多大な人的労力が必要
- インターネット関連企業やオープンソースソフトウェアのセキュリティが世界中に影響
- 防御的なセキュリティ作業の加速化が急務

### AI技術の進化

- Claude Mythos は従来のOpus/Sonnet/Haikuの階層を超越した新たなクラスのモデル
- 脆弱性の自動発見と悪用可能性の実証が技術的に実現可能に

### Project Glasswing による責任あるリリース

- AnthropicとAWSが採用する慎重なアプローチ
- インターネット関連企業とオープンソースメンテナーを優先
- 業界全体がセキュリティ向上の恩恵を受ける機会を創出

## 主な機能と改善

### 1. 脆弱性検出・実証機能

- **高度な脆弱性特定**: 従来のスキャンツールでは検出困難な複雑な脆弱性を自動発見
- **悪用可能性の実証**: 脆弱性が実際に悪用される可能性を検証
- **自動レポート生成**: 実行可能な知見を迅速に提供

### 2. コード理解能力

- **大規模コードベース対応**: 数百万行のコードを理解・分析
- **コンテキスト保持**: 複雑な依存関係やシステムアーキテクチャを把握
- **自動修正提案**: セキュリティ脆弱性の修正方法を提案

### 3. 防御的セキュリティ加速

- **自動化された脆弱性検査**: 人的ガイダンス最小化
- **継続的な監視**: 新規コミットやライブラリ更新の自動スキャン
- **業界知見共有**: セキュリティ対策の知見をコミュニティに還元

## 利用可能性と提供条件

### アクセス方法

- **提供方式**: Amazon Bedrock の Anthropic モデル欄から利用可能（許可リスト限定）
- **対象組織**: インターネット関連企業、オープンソースメンテナー、セキュリティ関連企業
- **連絡方法**: 許可リストに登録された組織のAWSアカウントチームが直接連絡
- **初期段階**: Project Glasswing の研究プレビューフェーズ

### 提供資源

- **利用クレジット**: Anthropic と AWS が合計で最大 $100 million の使用クレジットを提供
- **超過料金**: $100 million のクレジット超過後は通常の利用料金が適用
- **技術サポート**: AWSアカウントチームによる専任サポート

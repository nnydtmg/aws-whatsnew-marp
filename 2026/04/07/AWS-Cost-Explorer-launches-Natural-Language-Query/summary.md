# AWS Cost Explorer launches Natural Language Query capabilities powered by Amazon Q

**カテゴリ:** What's New
**公開日:** 2026-04-07T02:50:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/AWS-Cost-Explorer-Natural-Language-Query/](https://aws.amazon.com/about-aws/whats-new/2026/04/AWS-Cost-Explorer-Natural-Language-Query/)

---

## 要約

AWS Cost Explorerに自然言語クエリ機能が追加され、ユーザーはAmazon Qを通じて会話形式でコストデータを分析でき、自動的に更新される可視化により、より迅速で直感的なコスト分析が可能になりました。この機能は全ての商用AWSリージョンで追加料金なしで利用できます。

## 詳細

- AWS Cost Explorerに、Amazon Qの生成AIによる自然言語クエリ機能が統合されました。
- ユーザーは自然言語でAWS のコストと使用状況データについて質問できるようになりました。
- 質問に対する回答と同時に、Cost Explorerの可視化が自動的に更新されます。
- 「この月の上位支出サービスを表示してください」といった一般的なコスト質問の提案プロンプトが提供されます。
- 「Ask Question」ボタンを使用して、カスタム質問を自然言語で入力できます。
- コストと使用状況データに基づく分析では、チャートとテーブルが自動的に更新されます。
- 料金やアノマリー検出などの追加データセットからの分析結果は、Amazon Qのアーティファクトパネルに表示されます。
- 会話のコンテキストを保持したまま、フォローアップ質問を続行できます。
- この機能は全ての商用AWSリージョンで追加料金なしで利用可能です。
- コスト分析の時間を短縮し、すべてのチームメンバーにコスト可視性をもたらします。

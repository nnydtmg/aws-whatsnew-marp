# AWS Cost Explorer launches Natural Language Query capabilities powered by Amazon Q

**カテゴリ:** What's New
**公開日:** 2026-04-07T02:50:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-cost-explorer-natural-language-query/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-cost-explorer-natural-language-query/)

---

## 要約

AWS Cost Explorerに自然言語クエリ機能が追加され、ユーザーはAmazon Qを通じて自然言語でコストと使用状況データについて質問でき、自動的に更新されたビジュアライゼーションが表示されるようになりました。この機能は全ての商用AWSリージョンで追加料金なしで利用可能であり、コスト分析の効率化と全チームメンバーへのコスト可視化アクセスを実現します。

## 詳細

- AWS Cost Explorerに、Amazon Q Developerの生成AIによる自然言語クエリ機能が追加されました。
- ユーザーは自然言語を使用してAWS のコストと使用状況データについてAmazon Qに質問できるようになりました。
- 質問に対する回答と同時に、Cost Explorerで自動的に更新されたビジュアライゼーションが提供されます。
- 「Show me my top spending services for this month」などの一般的なコスト分析質問用の提案プロンプトが利用可能です。
- カスタム質問も「Ask Question」ボタンを使用して自由な形式で入力できます。
- コストと使用状況データに基づく分析の場合、Cost Explorerのチャートとテーブルが自動的に更新されます。
- 価格設定やアノマリ検出などの追加データセットからの分析結果は、Amazon Qの新しいアーティファクトパネルに表示されます。
- 会話のコンテキストを保持したまま、フォローアップ質問を続けることができます。
- この機能は全ての商用AWSリージョンで追加料金なしで利用可能です。
- この更新は、コスト分析の時間短縮、インサイト取得の迅速化、全チームメンバーへのコスト可視化アクセスを実現します。

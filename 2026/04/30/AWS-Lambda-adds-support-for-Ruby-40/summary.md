# AWS Lambda adds support for Ruby 4.0

**カテゴリ:** What's New
**公開日:** 2026-04-30T13:00:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-lambda-adds-ruby/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-lambda-adds-ruby/)

---

## 要約

AWS Lambda が Ruby 4.0 のサポートを追加し、マネージドランタイムとコンテナベースイメージとして利用可能になりました。2029年3月までのセキュリティサポートと JSON 構造化ログなどの高度なログ制御機能を備えており、最新の Ruby 言語機能を活用したサーバーレスアプリケーション開発に適しています。

## 詳細

- AWS Lambdaが Ruby 4.0 のサポートを追加しました。開発者はマネージドランタイムとコンテナベースイメージの両方として Ruby 4.0 を利用できます。

- AWS は自動的にマネージドランタイムとベースイメージの更新を適用します。

- Ruby 4.0 は最新の長期サポート（LTS）リリースであり、2029年3月までセキュリティとバグ修正がサポートされます。

- Lambda の高度なログ制御機能がサポートされており、JSON 構造化ログ、設定可能なログレベル、Amazon CloudWatch ロググループのターゲット設定が可能です。

- Ruby 4.0 ランタイムはすべての AWS リージョン（中国リージョンおよび AWS GovCloud（US）リージョンを含む）で利用可能です。

- Lambda コンソール、AWS CLI、AWS Serverless Application Model（AWS SAM）、CDK、AWS CloudFormation などのデプロイメントツールを使用して、Ruby 4.0 で作成されたサーバーレスアプリケーションをデプロイおよび管理できます。

- このアップデートは、最新の Ruby 言語機能へのアクセスと高度なログ管理機能を必要とする開発者に適しています。

- 長期サポートと自動更新により、エンタープライズレベルのアプリケーション開発に適しています。

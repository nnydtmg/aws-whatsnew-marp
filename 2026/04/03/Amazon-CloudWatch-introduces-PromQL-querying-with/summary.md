# Amazon CloudWatch introduces PromQL querying with Query Studio Preview

**カテゴリ:** What's New
**公開日:** 2026-04-03
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-cloudwatch-query-studio-preview/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-cloudwatch-query-studio-preview/)

---

## 要約

Amazon CloudWatchのQuery Studioは、PromQLとCloudWatch Metric Insightsを統合した新しいクエリおよび可視化機能であり、複数のコンソール間を切り替えることなく、AWSメトリクスとOpenTelemetryメトリクスを統一されたインターフェースで操作できます。このアップデートは、カスタムメトリクスとベンダー提供メトリクスを相関させて迅速に問題を特定したいアプリケーション開発チームおよびインフラストラクチャ管理者に特に有益です。

## 詳細

Amazon CloudWatchがQuery Studioをパブリックプレビューとして発表しました。Query Studioは、CloudWatchに初めてネイティブPromQLクエリ機能をもたらすものです。PromQLとCloudWatch Metric Insightsを単一のインターフェースに統合しています。AWSベンダー提供メトリクスとOpenTelemetryメトリクスの両方をクエリできます。ユーザーは複数のコンソール間を切り替えることなく、好みの言語でクエリできます。ビジュアルフォームビルダーとオートコンプリート機能が提供されています。コード エディタには構文ハイライト機能が搭載されています。新規ユーザーと経験豊富なユーザーの両方にアクセス可能です。Amazon EC2上で実行されるアプリケーションチームは、カスタムOpenTelemetryメトリクスとEC2ベンダー提供メトリクスを並べて相関させることができます。スタック全体の問題を素早く特定できます。クエリ結果から直接アラームを作成またはダッシュボードにチャートを追加できます。Query Studioは米国東部(バージニア北部)、米国西部(オレゴン)、アジアパシフィック(シドニー)、アジアパシフィック(シンガポール)、ヨーロッパ(アイルランド)で利用可能です。標準的なCloudWatchダッシュボード料金が適用されます。

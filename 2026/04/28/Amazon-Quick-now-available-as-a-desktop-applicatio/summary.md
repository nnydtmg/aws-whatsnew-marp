# Amazon Quick now available as a desktop application for macOS and Windows (Preview)

**カテゴリ:** What's New
**公開日:** 2026-04-28
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-quick-macos-windows-preview/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-quick-macos-windows-preview/)

---

## 要約

Amazon QuickのデスクトップアプリケーションがmacOSおよびWindowsでプレビュー版として利用可能になり、ローカルファイルアクセス、OS レベル通知、自動化機能などを備えています。このアップデートは、複数のアプリケーションやファイルにわたって統合された作業コンテキストを必要とするチームと個人ユーザーに特に適しています。

## 詳細

### 主要な新機能

1. **ネイティブデスクトップアプリケーション**
   - macOSおよびWindowsで利用可能（プレビュー版）
   - OS に最適化された操作性と応答性

2. **ローカルファイルへの直接アクセス**
   - ファイルをアップロードせずに読み取りと作業が可能
   - セキュリティとプライバシーを維持

3. **OS レベルのプロアクティブ通知**
   - アクションアイテムの追跡
   - カレンダーの競合を自動検出
   - メッセージで注意が必要な内容を優先表示

4. **ネイティブデスクトップコントロール**
   - ローカルファイルシステムとのシームレス統合
   - デスクトップ環境に最適化された体験

5. **ブラウザベースとデスクトップアプリの自動化**
   - 複数のアプリケーション環境を統合
   - ワークフロー全体の自動化が可能

6. **個人用ナレッジグラフ**
   - すべてのインタラクション全体で人、プロジェクト、関係を学習
   - 過去の文脈を自動的に把握
   - ワークフロー全体で一貫性のある支援

7. **開発者向けのコーディングエージェント**
   - ローカルModel Context Protocol（MCP）接続をサポート
   - ローカル開発環境でのエージェント実行
   - カスタムエージェントの構築が可能

8. **クロスプラットフォーム連携**
   - メモリ、ナレッジグラフ、エージェントはウェブとデスクトップ全体で共有
   - ブラウザとデスクトップアプリの無シームな統合

### 利用可能な範囲

- **対象ユーザー**: 米国東部（N. Virginia）のすべてのQuickサブスクライバー
- **ステータス**: プレビュー版
- **プラットフォーム**: macOS / Windows

### ユースケース

- **チームコラボレーション**: プロジェクトチームが複数のファイル、メール、カレンダーを管理
- **ナレッジワーカー**: 個人が多数の情報源から調査と分析を実施
- **開発・技術支援**: 開発者がコードレビューやデバッグを実施

## メリット

- **生産性向上**: 統合されたAIアシスタントによるワークフロー効率化
- **意思決定の加速**: 完全な作業コンテキストを理解したAIからの支援
- **時間削減**: 複数アプリケーション間でのコンテキスト切り替えを排除
- **セキュリティ**: ローカルファイルへの直接アクセスでセキュアに処理
- **開発生産性**: ブラウザタスクとデスクトップアプリケーション両方の自動化

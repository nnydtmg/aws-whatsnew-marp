# AWS Cost Optimization Hub now supports CSV download

**カテゴリ:** What's New
**公開日:** 2026-04-28
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-cost-optimization-hub-csv-download/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-cost-optimization-hub-csv-download/)

---

## 要約

AWS Cost Optimization Hubにおいて、コンソールから直接CSVファイルをワンクリックでダウンロードできる新機能が追加されました。この機能により、AWSコンソールへのアクセス権を持たないステークホルダーとの推奨事項の共有やオフライン環境での分析が容易になります。

## 詳細

AWS Cost Optimization Hubが、コンソールから直接CSVファイルをダウンロードする機能をサポートするようになりました。ワンクリックでローカルマシンにコスト最適化の推奨事項をエクスポートできるようになりました。既存のAmazon S3への自動エクスポート機能（Data Export）を補完する機能です。コンソールで適用されているフィルター、ソート設定、グループ化設定を反映した推奨事項をエクスポートできます。ダウンロードは即座に開始され、スプレッドシートアプリケーションでの分析が容易になります。AWSコンソールアクセス権を持たないステークホルダーとの推奨事項の共有が可能になります。オフライン環境や好みのツールでの推奨事項の作業が可能になります。この機能はAWS Cost Optimization Hubが提供されているすべてのリージョンで利用可能です。

## 主な特徴

### 機能面

- **直感的**: ワンクリックでコンソール内からのダウンロード
- **スマート**: フィルター、ソート、グループ化を反映
- **即座**: ダウンロード完了直後から分析可能
- **柔軟**: 既存の自動エクスポート機能と併用可能

### 利用シーン

- 経営層への報告資料作成
- FinOps チームでの詳細分析
- マルチアカウント管理
- オフライン環境での検討

## ビジネスインパクト

### 効果

1. **運用効率化**: コンソールアクセス権不要でのデータ共有
2. **分析の民主化**: 好みのツールでの分析が可能
3. **意思決定の加速**: スプレッドシートでの即座な分析
4. **コスト削減**: 推奨事項の迅速な実装

### 対象ユーザー

- FinOps チーム
- 経営層・意思決定者
- AWS コンソールアクセス権のないステークホルダー
- コスト最適化プログラムの実施者

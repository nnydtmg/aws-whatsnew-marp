# Build custom applications using natural language in Amazon Quick (Preview)

**カテゴリ:** What's New
**公開日:** 2026-04-28
**元記事:** https://aws.amazon.com/about-aws/whats-new/2026/04/custom-applications/

---

## 要約

Amazon Quickの新機能は、技術スキルがないユーザーが自然言語でカスタムウェブアプリケーションを迅速に構築できるようにするものであり、営業リーダーや財務マネージャーなどのビジネスユーザーが自分たちの業務効率化を実現するのに適しています。

---

## 詳細

### 主な特徴

- **自然言語アプリケーション構築**: Amazon Quickは自然言語を使用してカスタムウェブアプリケーションを数分で作成できる新機能をプレビューで発表
- **ノーコード開発**: コーディング知識がなくても、ユーザーが必要な内容を説明するだけで、完全にインタラクティブなアプリケーションを取得可能
- **高度な統合機能**: 作成されたアプリケーションはライブデータソースに接続し、複雑なワークフローを実装し、AI機能を組み込む
- **簡単な公開・共有**: ワンクリックでアプリケーションを公開し、チームと共有することが可能

### ユースケース

#### 営業向け
- CRMやビジネスアプリケーションからリアルタイムでパイプラインレビュー用アプリケーションを作成
- 営業メンバーのパフォーマンス可視化

#### 財務向け
- QuickBooks、Excel、内部システムから情報を集約
- 月次決算を簡素化
- 財務レポート自動生成

### 利用開始の容易性

- **無料**: Amazon Quickは無料で利用でき、AWSアカウントやクレジットカードは不要
- **高速オンボーディング**: ガイド付きオンボーディング体験により、5分以内に価値を見出すことが可能
- **多様なテンプレート**: 営業、マーケティング、財務、人事などの役職別ワークフローが提供されている

---

## 影響

### ビジネス価値

1. **生産性向上**: 複数システムのデータ統合にかかる時間を大幅削減
2. **コスト最適化**: 無料で利用可能、複数ツール購入が不要
3. **デジタル人材育成**: ビジネスユーザーが自律的に問題解決できるスキル獲得
4. **IT部門の負担軽減**: 単純な業務自動化要件を削減

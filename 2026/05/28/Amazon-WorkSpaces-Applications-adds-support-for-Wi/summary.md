# Amazon WorkSpaces ApplicationsがWindows Desktop OSをサポート

Amazon WorkSpaces Applications adds support for Windows Desktop OS

**カテゴリ:** What's New
**公開日:** 2026-05-28
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-workspaces-applications-windows-desktop-OS/)

このページでは、AWS What's Newで発表された「Amazon WorkSpaces Applications adds support for Windows Desktop OS」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

Amazon WorkSpaces ApplicationsがWindows Desktop OSのBYOL対応を開始し、お客様は既存のWindows Desktopライセンスを持ち込むことで、OS費用を削減しながら一貫したデスクトップ体験を提供できるようになりました。

## このアップデートで何が変わったか

- Amazon WorkSpaces Applicationsが、Windows Desktop OSを使用したストリーミングリソースのセットアップに対応
- Bring Your Own License（BYOL）により、既存のWindows Desktopライセンスを持ち込むことが可能
- Microsoft 365 Apps for enterpriseの対象ライセンスに対応
- OSはお客様のAWSアカウント専用のハードウェア上でホスト
- 既存ライセンスを持ち込むことでOS費用を削減
- ローカルとリモートで同じワークフロー、ショートカット、ナビゲーションが利用可能
- 複数のAWSリージョンで利用可能

## 対象ユーザー

- オンプレミスと仮想デスクトップ環境間でシームレスな作業環境を求める企業
- Windows Desktop OSライセンスを既に保有している組織
- Microsoft 365 Apps for enterpriseを利用している企業
- OS費用の削減を目指す組織

## 主なメリット

1. **OS費用の削減**: BYOLにより、OS費用を排除して、コンピュート費用とストリーミングインフラストラクチャ費用のみの支払いが可能
2. **一貫性のあるデスクトップ体験**: ローカルとリモート環境で同じワークフロー、ショートカット、ナビゲーションが利用可能
3. **オンボーディング時間の短縮**: 異なるデスクトップ体験に適応する認知的オーバーヘッドが削減
4. **エンタープライズ対応**: Microsoft 365 Apps for enterpriseの対象ライセンスで対応
5. **スケーラビリティ**: Windowsデスクトップアプリケーションとフルデスクトップ体験をスケーリング可能

## 詳細

### Bring Your Own License（BYOL）について

- 既存のWindows Desktop OSライセンスを持ち込むことで、OS費用を削減
- Microsoftのライセンス要件を満たし、毎月特定のAWSリージョンで最小数のストリーミングリソースを実行することをコミットする必要がある

### 対応リージョン

- Windows Desktop for WorkSpaces Applicationsは複数のAWSリージョンで利用可能

### ハードウェア構成

- OSはお客様のAWSアカウント専用のハードウェア上でホスト
- 組織のAWSアカウント内で運用される

## ユースケース

### ハイブリッドワーク環境

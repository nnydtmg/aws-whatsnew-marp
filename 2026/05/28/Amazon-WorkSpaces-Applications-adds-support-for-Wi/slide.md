# Amazon WorkSpaces ApplicationsがWindows Desktop OSをサポート

Amazon WorkSpaces Applications adds support for Windows Desktop OS

**What's New** | 2026-05-28

---

## 概要

- Amazon WorkSpaces ApplicationsがWindows Desktop OSのBYOL対応を開始し、お客様は既存のWindows Desktopライセンスを持ち込むことで、OS費用を削減しながら一貫したデスクトップ体験を提供できるようになりました。

## 変更内容・新機能

- Amazon WorkSpaces Applicationsが、Windows Desktop OSを使用したストリーミングリソースのセットアップに対応
- Bring Your Own License（BYOL）により、既存のWindows Desktopライセンスを持ち込むことが可能
- Microsoft 365 Apps for enterpriseの対象ライセンスに対応
- OSはお客様のAWSアカウント専用のハードウェア上でホスト
- 既存ライセンスを持ち込むことでOS費用を削減
- ローカルとリモートで同じワークフロー、ショートカット、ナビゲーションが利用可能
- 複数のAWSリージョンで利用可能

---

## 効果・メリット

- **OS費用の削減**: BYOLにより、OS費用を排除して、コンピュート費用とストリーミングインフラストラクチャ費用のみの支払いが可能
- **一貫性のあるデスクトップ体験**: ローカルとリモート環境で同じワークフロー、ショートカット、ナビゲーションが利用可能
- **オンボーディング時間の短縮**: 異なるデスクトップ体験に適応する認知的オーバーヘッドが削減
- **エンタープライズ対応**: Microsoft 365 Apps for enterpriseの対象ライセンスで対応
- **スケーラビリティ**: Windowsデスクトップアプリケーションとフルデスクトップ体験をスケーリング可能

---

## ユースケース

- **ハイブリッドワーク環境**: オンプレミスと仮想デスクトップ環境間でシームレスな作業環境を提供
- **コスト最適化**: 既存のWindows Desktopライセンスを活用してOS費用を削減
- **ライセンス統一**: 複数のAWSリージョンでMicrosoft 365ライセンスを統一できる環境構築

# Claude Opus 4.8 が AWS で利用開始

Claude Opus 4.8 is now available on AWS

**What's New** | 2026-05-28

---

## 概要

- Claude Opus 4.8がAWSで利用可能になり、エージェンティックコーディング、専門知識作業、自律タスクにおいて高度な推論能力と信頼性を提供いたします。
- 開発者とエンタープライズは、Amazon BedrockまたはClaude Platform on AWSを通じてこの最先端モデルにアクセスできるようになりました。

---

## 前提・背景

### 市場の動向と課題

- エンタープライズの複雑な自動化タスク需要の拡大
- エージェンティックAIの信頼性と推論能力に対する期待の高まり
- コーディング、データ分析、意思決定支援の自動化ニーズ

### Anthropic の進化

- Opus 4.7 (2026年4月) からさらに性能向上
- 本番環境での信頼性と一貫性が大幅改善
- 長時間実行タスクへの対応強化

---

## 変更内容・新機能

### 1. 主要な改善

- **より長い自律実行**: 複雑なマルチステップタスクを継続的に実行
- **より深い推論**: 複雑な問題解決能力の向上
- **本番環境での信頼性**: 一貫性と予測可能性の強化

### 2. コーディング機能

- エンジニアのようにコードベースを読解
- 編集前に計画立案
- 長いセッション中のコンテキスト保持

### 3. エージェンティックタスク

- 障害物を回避する能力
- 自身のエラーから回復
- 支援を求めるべき時と継続すべき時の判断

---

## 利用方法

### Amazon Bedrock での利用

- データはAWSインフラストラクチャ内に保持
- Guardrails、Knowledge BasesなどのAWS管理機能を利用可能
- 地域データレジデンシーをサポート

### Claude Platform on AWS での利用

- AWSコンソール経由でAnthropicのネイティブプラットフォーム体験を提供
- AWS課金と認証で統一
- 管理の簡素化

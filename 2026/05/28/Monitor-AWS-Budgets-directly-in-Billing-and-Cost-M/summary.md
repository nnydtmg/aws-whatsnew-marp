# AWS Billing and Cost Management ダッシュボードに新しい Budgets ウィジェットが登場

**英語タイトル:** Monitor AWS Budgets directly in Billing and Cost Management Dashboards with new Budgets widget

**カテゴリ:** What's New  
**公開日:** 2026年5月28日  
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/monitor-aws-budgets-using-dashboards)

---

## 📋 要約

AWS Billing and Cost Management（BCM）ダッシュボードに **Budgets ウィジェット** が新たに追加されました。このウィジェットにより、ユーザーは予算パフォーマンス、実際の支出、予測額を**単一のダッシュボード内で監視**できるようになりました。

従来は予算データを確認するために BCM コンソール内で複数ページを行き来する必要がありましたが、この新機能により運用効率が大幅に改善されます。

---

## ✨ このアップデートで何が変わったか

### 新機能: Budgets ウィジェット

- ✅ **BCM ダッシュボードに統合** - 別途のコンソールページアクセスが不要に
- ✅ **ワンクリック監視** - 予算パフォーマンス、実際支出、予測額を一箇所で確認
- ✅ **フィルタリング機能** - 名前、閾値、予算タイプで絞り込み可能
- ✅ **レポート統合** - ダッシュボードのエクスポート機能と完全統合
- ✅ **追加料金なし** - 全 AWS コマーシャルリージョンで無料

## 📊 主な機能と仕様

### Budgets ウィジェットの表示項目

| 項目 | 説明 |
|------|------|
| 予算名 | ウィジェット内で一覧表示 |
| 予算額 | 設定された予算の金額 |
| 実際の支出 | 現在の使用額 |
| 予測額 | 月末までの見積もり支出 |

### フィルタリング・操作機能

- **名前でフィルタリング** - 特定の予算を表示
- **閾値でフィルタリング** - 特定の達成率の予算を表示
- **予算タイプでフィルタリング** - リソースグループベース、キーベースなど
- **カスタムビュー作成** - 自社の監視要件に合わせたビュー

### レポート・エクスポート機能

- ✅ **ダッシュボード統合** - ダッシュボード全体のエクスポート機能と統合
- ✅ **スケジュール済みメール配信** - 日次・週次・月次で自動配信
- ✅ **複数形式対応** - CSV、PDF の両形式でダウンロード可能
- ✅ **安全な配信** - パスワード保護された PDF レポート

## 💼 ユースケース・活用例

### 財務チームの運用

```
従来の方法:
 BCM コンソール → Budgets ページ → 予算確認 → Cost Explorer ページ → 支出確認
  ↓ (ページ遷移に時間)

新しい方法:
 BCM ダッシュボード → Budgets ウィジェット + Cost Explorer ウィジェット → 一元監視
  ↓ (効率化)
```

   - 本番環境、開発環境、テスト環境で分けて監視
   - 部門別、プロジェクト別で管理

2. **他のウィジェットと並べて表示**
   - Cost Explorer レポート
   - Savings Plans カバレッジ
   - Reserved Instances 利用率

3. **スケジュール済みレポートで定期共有**
   - 月次レポートを CFO に自動配信
   - 週次チェックシートをチームに配信

## 🚀 始め方

### ステップバイステップガイド

#### 1. AWS Management Console にログイン

```
1. AWS Management Console を開く
2. アカウント ID またはメールアドレスでログイン
```

#### 2. Billing and Cost Management コンソール へ移動

```
1. コンソールホーム → [Billing and Cost Management] をクリック
2. または、検索ボックスで「Billing」と入力
```

#### 3. Dashboards セクションを開く

```
1. 左ナビゲーションパネルで [Dashboards] を選択
2. 既存ダッシュボードの一覧が表示
```

#### 4. Budgets ウィジェットを追加

```
1. 既存のダッシュボードを選択、または新規作成
2. [ウィジェット追加] ボタンをクリック
3. [Budgets] を選択
4. フィルタリング条件を必要に応じて設定
```

#### 5. ダッシュボードにピン留め

```
1. ウィジェットの右上にあるピンアイコンをクリック
2. ダッシュボードに確定
```

### 活用のヒント

1. **複数ウィジェットの活用**
   - 部門別に異なる Budgets ウィジェットを作成
   - 階層的な予算管理に対応

2. **他のウィジェットとの組み合わせ**
   - Cost Explorer ウィジェットと並べて表示
   - Savings Plans カバレッジと一緒に監視

3. **スケジュール済みレポートの活用**
   - 月次レポートで CFO に定期報告
   - 週次チェックシートでチーム共有

4. **フィルタリングの工夫**
   - 重要な予算に絞った表示
   - 達成率ごとのビュー分けなど

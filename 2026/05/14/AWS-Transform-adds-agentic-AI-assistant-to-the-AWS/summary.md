# AWS Transform が Visual Studio 用 AWS Toolkit にエージェント型 AI アシスタントを追加

AWS Transform adds agentic AI assistant to the AWS Toolkit for Visual Studio

**カテゴリ:** What's New
**公開日:** 2026-05-14
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-transform-ai-assistant)

このページでは、AWS What's Newで発表された「AWS Transform adds agentic AI assistant to the AWS Toolkit for Visual Studio」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

AWS Transform は Visual Studio 用 AWS Toolkit にエージェント型 AI アシスタント機能を追加し、.NET 開発者が IDE 内で対話的にアプリケーション最新化を実行できるようになりました。このアップデートは、IDE から離れることなく段階的な制御と透明性を備えた最新化プロセスを求める .NET 開発者に最適です。

## このアップデートで何が変わったか

- AWS Transform に、Visual Studio 用 AWS Toolkit 内でエージェント型 AI アシスタントが追加されました。
- .NET 開発者は、IDE 内で対話的かつ段階的なガイダンスを通じてアプリケーションの最新化を実現できます。
- ソースコード分析、詳細な評価レポート生成、変換計画の作成が可能です。
- 開発者は各ステップを確認、編集、承認してから進めることができ、Web コンソールに切り替える必要がありません。
- ビルドエラーの自動修正、詳細なワークログ提供、完了時の HTML サマリーレポート生成が特徴です。
- Web コンソールで開始したプロジェクトを Visual Studio で継続でき、コンテキストと進捗が保持されます。
- Kiro や他の AI コーディングアシスタントからも AWS Transform エージェントを利用できます。
- 8 つの AWS リージョン（米国東部、カナダ中部、ヨーロッパ 2 地域、アジアパシフィック 4 地域）で利用可能です。

## 詳細

### エージェント型 AI アシスタント機能

- ソースコード分析を自動実施
- 詳細な評価レポートを生成
- 段階的な変換計画を作成
- 対話的なガイダンスで各ステップを確認・承認可能

### 統合機能

- ビルドエラーの自動検出と修正
- 詳細なワークログの記録
- 完了時の HTML サマリーレポート生成

### クロスツール対応

- Web コンソールとの連携
- Kiro や他の AI コーディングアシスタント対応
- VS 2022 および VS 2026 対応

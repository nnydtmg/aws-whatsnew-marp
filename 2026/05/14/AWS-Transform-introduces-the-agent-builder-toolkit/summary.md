# AWS Transform エージェントビルダーツールキット「Kiro power」が一般提供開始

AWS Transform introduces the agent builder toolkit Kiro power for building customized transformation agents

**カテゴリ:** What's New
**公開日:** 2026-05-14
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-transform-agent-builder-toolkit/)

このページでは、AWS What's Newで発表された「AWS Transform introduces the agent builder toolkit Kiro power for building customized transformation agents」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

AWS Transformは、カスタマイズされたトランスフォーメーションエージェントを構築・共有・登録できるエージェントビルダーツールキット「Kiro power」の一般提供を開始しました。このツールキットにより、パートナーおよび顧客は、AWS Transformのエージェンティック AI機能とシームレスに統合された、差別化されたモダナイゼーションソリューションを構築することが可能になります。

---

## このアップデートで何が変わったか

### 主な変更点

1. **エージェントビルダーツールキット「Kiro power」の一般提供開始**
   - AWSパートナーおよび顧客が、特定のモダナイゼーション要件に合わせてカスタマイズされたエージェントを構築可能
   - 構築したエージェントはAWS Transformのエージェンティック AI機能とシームレスに統合

2. **エンドツーエンドライフサイクルの提供**
   - エージェント構築
   - チーム間またはパートナーネットワーク全体での共有
   - AWS Transformへの登録
   - Kiro powerマーケットプレイスでの検出

3. **差別化ソリューション構築の実現**
   - Migration and Modernization Competency Partners
   - 独立系ソフトウェアベンダー（ISV）
   - AWSの顧客
   - 上記のユーザーが専門化されたエージェント、ツール、ナレッジベース、ワークフローを統合

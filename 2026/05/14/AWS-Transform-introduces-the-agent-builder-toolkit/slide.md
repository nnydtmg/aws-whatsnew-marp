# AWS Transform エージェントビルダーツールキット「Kiro power」が一般提供開始

AWS Transform introduces the agent builder toolkit Kiro power for building customized transformation agents

**What's New** | 2026-05-14

---

## 概要

- AWS Transformは、カスタマイズされたトランスフォーメーションエージェントを構築・共有・登録できるエージェントビルダーツールキット「Kiro power」の一般提供を開始しました。
- このツールキットにより、パートナーおよび顧客は、AWS Transformのエージェンティック AI機能とシームレスに統合された、差別化されたモダナイゼーションソリューションを構築することが可能になります。

---

## 前提・背景

### 市場の動向と課題

- エンタープライズのモダナイゼーション需要の増加
- レガシーシステムから最新テクノロジーへの移行ニーズ
- カスタマイズ可能なAIエージェントの必要性

### AWS Transformの進化

- **April 2026**: AWS Transform が Kiro と VS Code で利用可能に
- **May 2026**: コンテナ化機能の追加
- エージェンティック AI による自動化の強化

---

## 主な機能と改善

### 1. エージェントビルダーツールキット

- **カスタマイズ可能**: 特定のモダナイゼーション要件に合わせたエージェント構築
- **統合**: AWS Transform のエージェンティック AI 機能とシームレスに統合
- **差別化**: 専門化されたエージェント、ツール、ナレッジベース、ワークフローを統合

### 2. エンドツーエンドライフサイクル

- エージェント構築
- チーム間およびパートナーネットワーク全体での共有
- AWS Transform への登録
- Kiro power マーケットプレイスでの検出

# Amazon WorkSpaces now lets AI agents operate desktop applications (Preview)

**カテゴリ:** What's New  
**公開日:** 2026-05-05  
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/05/workspaces-ai-agents/](https://aws.amazon.com/about-aws/whats-new/2026/05/workspaces-ai-agents/)

---

## 要約

Amazon WorkSpacesは、AIエージェントがモダンAPIを持たないレガシーデスクトップアプリケーションを安全に操作・自動化できるようにする新機能をプレビューで提供開始いたしました。エンタープライズグレードのガバナンス、コンプライアンス、可視性を維持しながら、金融サービスやヘルスケアなどの規制対象産業における複雑なワークフロー自動化を実現いたします。

---

## 詳細

### 新機能の概要

- Amazon WorkSpacesがAIエージェントによるデスクトップアプリケーション操作を可能にする新機能がプレビューで利用可能になりました。

- モダンAPIを持たないメインフレーム、ERPシステム、独自ツールなどのレガシーアプリケーションをAIエージェントが自動化できるようになりました。

### 技術的特徴

- 業界標準のModel Context Protocol（MCP）統合により、最小限のコードでクラウド、オンプレミス、ハイブリッド環境からビジネスアプリケーションに接続できます。

- IT管理者は人間のWorkSpaces環境と同じ一元化された権限管理、ログ記録、監査制御を維持できます。

### エンタープライズ機能

- スクリーンショットとメトリクスを含むエンタープライズ可視性機能により、エージェント活動の完全な可視化が実現します。

### 適用可能なワークフロー

- 請求処理、取引決済、候補者スクリーニング、バックオフィス業務などのワークフロー自動化が可能になります。

### 対応業界

- 金融サービス、ヘルスケア、その他の規制対象産業でアプリケーション近代化を必要とせずに実装できます。

### コスト効率

- 従量課金制と弾力的なスケーリングにより、IT運用負荷を軽減しながら人間とAIの協働を拡大できます。

### 直接的なメリット

1. **時間短縮**: 手作業の自動化による処理時間削減

3. **エラー削減**: 自動化による人為的ミスの排除

3. **柔軟性**: AWS全体の機能と統合可能

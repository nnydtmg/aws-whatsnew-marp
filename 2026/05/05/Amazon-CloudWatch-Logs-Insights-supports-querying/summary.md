# Amazon CloudWatch Logs Insights supports querying by log group tags

**カテゴリ:** What's New
**公開日:** 2026-05-04T23:00:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-cloudwatch-logs-query-by-tags/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-cloudwatch-logs-query-by-tags/)

---

## 要約

Amazon CloudWatch Logs Insightsは、ロググループをタグを使用してクエリできる新機能をリリースしました。この機能により、顧客は共通のタグを持つすべてのロググループ全体でクエリを実行でき、環境の成長に伴う運用オーバーヘッドを削減できます。

## 詳細

- Amazon CloudWatch Logs Insightsがロググループタグを使用したクエリに対応しました。

- ロググループ名、データソース、ファセットに加えて、タグを使用したクエリが可能になりました。

- タグはEnvironment: ProductionやApplication: PaymentServiceなど、ロググループを分類するためのキーバリューペアです。

- 共通のタグを持つすべてのロググループ全体でクエリを実行できるようになりました。

- ロググループタグの追加または削除時に、クエリが自動的にマッチするロググループを反映します。

- 環境が成長する際の運用オーバーヘッドが削減されます。

- この機能はすべての商用AWSリージョンで利用可能です。

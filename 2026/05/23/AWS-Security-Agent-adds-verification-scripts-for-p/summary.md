# AWS Security Agent がペネトレーション検査検出結果の検証スクリプト自動生成をサポート

AWS Security Agent adds verification scripts for pentest findings

**カテゴリ:** What's New
**公開日:** 2026-05-22T17:21:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-security-agent/)

## 要約

AWS Security Agentの新機能により、ペネトレーションテストの検出結果に対して検証スクリプトが自動生成されるようになり、セキュリティチームが脆弱性を独立して検証できるようになりました。この機能はトリアージプロセスの効率化と修復の加速化を実現いたします。

## このアップデートで何が変わったか

- AWS Security Agentは、ペネトレーションテストの検出結果に対して検証スクリプトを自動生成する新機能を追加いたしました。

- AWS Security AgentがサポートされているすべてのAWSリージョンで利用可能です。

## 対象ユーザー

- セキュリティチームは、生成された検証スクリプトをダウンロードして環境変数を設定し、対象システムに対して実行することで、脆弱性を独立して再現・検証することができるようになりました。

## 詳細

- AWS Security Agentは、ペネトレーションテストの検出結果に対して検証スクリプトを自動生成する新機能を追加いたしました。

- セキュリティチームは、生成された検証スクリプトをダウンロードして環境変数を設定し、対象システムに対して実行することで、脆弱性を独立して再現・検証することができるようになりました。

- 検証スクリプトには、セットアップ手順、ドキュメント化された環境変数、および機密値がマスクされた状態で含まれております。

- この機能により、チームは手動で検出結果の詳細から再現手順を追跡する必要がなくなり、トリアージの効率化と修復の加速化が実現いたします。

- AWS Security AgentがサポートされているすべてのAWSリージョンで利用可能です。

- ペネトレーションテストを実行し、検出結果に移動して検証スクリプトセクションを展開することで、この機能の利用を開始できます。

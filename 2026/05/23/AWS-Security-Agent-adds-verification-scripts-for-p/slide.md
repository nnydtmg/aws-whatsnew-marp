# AWS Security Agent がペネトレーション検査検出結果の検証スクリプト自動生成をサポート

AWS Security Agent adds verification scripts for pentest findings

**What's New** | 2026-05-22T17:21:00

---

## 概要

- AWS Security Agentの新機能により、ペネトレーションテストの検出結果に対して検証スクリプトが自動生成されるようになり、セキュリティチームが脆弱性を独立して検証できるようになりました。
- この機能はトリアージプロセスの効率化と修復の加速化を実現いたします。

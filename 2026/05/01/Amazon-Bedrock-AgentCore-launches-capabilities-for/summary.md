# Amazon Bedrock AgentCore launches capabilities for optimizing agent performance in preview

**カテゴリ:** What's New
**公開日:** 2026-04-30T15:00:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/05/bedrock-agentcore-optimization-preview/](https://aws.amazon.com/about-aws/whats-new/2026/05/bedrock-agentcore-optimization-preview/)

---

## 要約

Amazon Bedrock AgentCoreが推奨事項、バッチ評価、A/Bテストの3つの新機能をプレビューで提供開始し、本番環境でのAIエージェントの性能最適化を体系的に実施できるようになりました。この更新は、エージェントのパフォーマンス低下に対処し、検証済みの改善を段階的に展開したい開発チームに最適です。

## 詳細

- Amazon Bedrock AgentCoreが、エージェントのパフォーマンス最適化のための新機能をプレビューで提供開始いたしました。

- 新機能には、推奨事項（recommendations）、バッチ評価（batch evaluations）、A/Bテストの3つの機能が含まれております。

- これらの機能により、観察、評価、改善というAIエージェントの本番運用ループが完成いたしました。

- 推奨事項機能は、本番トレースと評価出力を分析して、ワークロードに最適化されたシステムプロンプトとツール説明を生成いたします。

- バッチ評価は、事前定義されたテストケースに対して推奨事項を検証するために使用されます。

- A/Bテストは、事前定義されたテストセットまたは本番トラフィックに対して推奨事項を検証し、統計的有意性を報告いたします。

- すべての推奨事項は、本番環境への展開前に開発者の承認が必要となります。

- この最適化機能は、AgentCore評価が利用可能なすべてのAWSリージョンで使用できます。

- この更新は、手動による開発者の介入と直感に頼らず、体系的なアプローチでエージェントの品質改善を行いたい組織に適しております。

- 本番運用中にエージェントの品質が時間とともに低下することに対処したい開発チームに特に有用です。

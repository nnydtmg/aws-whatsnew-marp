# AWS Transform now offers BI migration agents for Power BI and Tableau to Amazon QuickSight

**カテゴリ:** What's New
**公開日:** 2026-05-01
**元記事:** https://aws.amazon.com/about-aws/whats-new/2026/05/quick-bi-migration/

---

## 要約

AWS Transformは、Power BIおよびTableauダッシュボードをAmazon QuickSightに変換するBI移行エージェントを新たに提供し、移行期間を数ヶ月から数日に短縮することができるようになりました。このソリューションは、エンタープライズのBI環境を迅速にクラウドへ移行したい組織に最適です。

---

## 詳細

### 新機能

- **AWS Transformの新しいBI移行エージェント**: Power BIおよびTableauダッシュボードをAmazon QuickSightアセットに変換するエージェントを提供

- **移行期間の大幅短縮**: これまで数ヶ月要していた移行を数日で実現

- **Wavicle Data Solutionsとの提携**: AWS Advanced Consulting Partnerとして、AWS Transformのエージェンティックなアイ機能と統合

### エージェント構成

- **4つのエージェント**: Power BI用とTableau用それぞれのAnalyzerエージェントとConverterエージェント
- **AWS Marketplace経由での購入**: 専門ソリューション提供者から直接購入可能

### 機能詳細

#### チャットベースのインターフェース
- ソースダッシュボードの移行準備状況を評価
- データセット、計算フィールド、ビジュアライゼーション、フィルターをAmazon QuickSightで自動再構築

#### セキュリティ
- すべての処理がお客様のAWSアカウント内で実行
- データは環境外に出ない

#### AI搭載機能
- 移行後、チームはAmazon QuickSightのAI搭載ワークフロー（自然言語ビジネス質問、自動化された調査、データ駆動型アクション）を活用可能

### 可用性

- **現在**: US East (N. Virginia)で利用可能
- **対応リージョン**: Amazon QuickSightが利用可能なすべての商用リージョンでQuickSightアセットの作成をサポート

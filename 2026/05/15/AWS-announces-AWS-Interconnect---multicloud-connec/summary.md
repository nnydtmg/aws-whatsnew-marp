# AWS Interconnect - multicloud が Oracle Cloud Infrastructure への対応をプレビュー開始

AWS announces AWS Interconnect - multicloud connectivity with Oracle Cloud Infrastructure in preview

**カテゴリ:** What's New
**公開日:** 2026-05-15T15:00:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-announces-AWS-interconnect-multicloud-oci-preview/)

このページでは、AWS What's Newで発表された「AWS announces AWS Interconnect - multicloud connectivity with Oracle Cloud Infrastructure in preview」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

AWS Interconnect - multicloud は、AWS と OCI 間のシームレスで復元力のあるマルチクラウド接続を提供する新機能であり、複数のクラウドプロバイダー間でのワークロード相互接続を必要とする顧客に最適です。本機能により、従来の複雑なマルチクラウド構築プロセスが大幅に簡素化されます。

## このアップデートで何が変わったか

- AWS Interconnect - multicloud は、AWS と Oracle Cloud Infrastructure (OCI) 間のマルチクラウド接続を実現する新機能です。
- この機能により、顧客は複数のクラウドサービスプロバイダー間で、復元力があり、スケーラブルなプライベート接続を迅速にプロビジョニングできます。
- AWS Interconnect は、Google Cloud (一般提供中) と Microsoft Azure (2026年後半予定) に加えて、OCI (プレビュー) でも利用可能になりました。
- 本機能は、マルチクラウド戦略を採用し、複数のクラウドプロバイダー間でワークロードを相互接続する必要がある顧客に適しています。
- 従来の「自力でのマルチクラウド」アプローチの複雑性を排除し、グローバルなマルチレイヤーネットワークの構築と管理を簡素化します。
- AWS Management Console、CLI、または API を使用して、プレビュー版の Interconnect を作成できます。
- 現在、us-east-1 (N. Virginia) AWS リージョンで利用可能です。

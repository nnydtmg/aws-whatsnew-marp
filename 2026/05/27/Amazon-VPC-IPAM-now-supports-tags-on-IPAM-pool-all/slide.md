# Amazon VPC IPAM のプール割り当てがタグ付けに対応

Amazon VPC IPAM now supports tags on IPAM pool allocations

**What's New** | 2026-05-26T13:00:00

---

## 概要

- Amazon VPC IPAM が IPAM プール割り当てへのタグ付けをサポートするようになり、顧客は AWS の他のリソースと同じタグ付けワークフローを使用して IP アドレス割り当てを整理、管理、およびアクセス制御できるようになりました。
- この機能により、IAM ポリシーを通じた一元的なガバナンスと、マルチアカウント環境での効率的な IP アドレス範囲の検索が可能になります。

## 変更内容・新機能

- 顧客は、IP アドレス割り当ての作成時にタグを付与することも、既存の割り当てにタグを追加することも可能になりました。

- この機能は、IPAM が利用可能なすべての AWS リージョンで、追加コストなしで利用可能です。

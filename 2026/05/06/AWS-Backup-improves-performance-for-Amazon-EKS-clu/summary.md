# AWS Backup improves performance for Amazon EKS cluster backups

**カテゴリ:** What's New
**公開日:** 2026-05-05T10:00:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-backup-amazon-eks-performance-improvement/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-backup-amazon-eks-performance-improvement/)

---

## 要約

AWS Backupは、Amazon EKSクラスターのバックアップ性能を最大10倍向上させ、大規模クラスターのバックアップ時間を数日から数時間に短縮することを発表しました。この性能改善は、すべてのAWSリージョンで自動的に有効化され、追加費用なしで利用できます。

## 詳細

- AWS Backupは、Amazon EKSクラスターのバックアップ性能を最大10倍高速化したことが発表されました。

- 大規模なネームスペースとKubernetesリソースを持つAmazon EKSクラスターのバックアップ時間が、数日から数時間に短縮されます。

- この性能改善は、すべてのAWSリージョンにおいて自動的に有効化され、追加費用は発生しません。

- AWS Backupは、Amazon EKSおよびコンピュート、ストレージ、データベースなどの他のAWSサービスのデータ保護を一元化および自動化するポリシーベースの完全マネージドソリューションです。

- このアップデートは、大規模なAmazon EKSクラスターを運用し、迅速なバックアップ完了を必要とする組織に適しています。

- AWS Backupサポートは、すべてのAWSコマーシャルリージョンおよびAWS GovCloud（US）リージョンで利用可能です。

# AWS Console Mobile App adds interactive graphs, AI log summaries, and natural language logs search to CloudWatch Alarms

**カテゴリ:** What's New  
**公開日:** 2026-05-04  
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-console-mobile-app/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-console-mobile-app/)

---

## 要約

AWS Console Mobile Appに、CloudWatch Alarmsの調査を効率化するためのインタラクティブグラフ、AI生成ログサマリー、自然言語ログ検索などの新機能が追加されました。これらの機能により、エンジニアやオペレーション担当者は、通知から根本原因の特定までの時間を大幅に短縮できるようになります。

---

## 詳細

### 追加された新機能

#### 1. インタラクティブグラフ
- アラームをトリガーしたメトリクスを可視化
- 特定の時間枠にズームインして異常を迅速に特定
- CloudWatch Dashboardスクリーンへのリンク対応

#### 2. AI生成ログサマリー
- 主要な要因を強調したログの概要を自動生成
- 複雑なログから重要情報を素早く抽出

#### 3. 自然言語ログ検索
- テキスト入力によるログ検索
- 音声入力によるハンズフリー検索
- 保存済みのLogs Insightsクエリの活用

#### 4. 運用機能の拡張
- タイムセレクターでカスタム時間範囲を表示
- タイムゾーンを運用ニーズに合わせて調整
- 関連するメトリクスとリソースをアラーム画面に表示

### メリット

- **時間短縮**: 通知から根本原因特定までの時間を大幅に短縮
- **効率化**: 手動でのログ検索作業を削減
- **モバイル対応**: 外出先からの迅速なトラブルシューティング
- **利便性向上**: 音声検索による利便性の向上

### 対応範囲

- すべてのAWS Commercial Regionsで利用可能
- 追加費用なし
- AWS Console Mobile Appのダウンロードで利用開始

### 適用対象

#### エンジニア・オペレーション担当者
- CloudWatch Alarmsを頻繁に調査する必要がある
- 迅速な問題解決が求められる

#### DevOpsチーム
- モバイルデバイスからの迅速な問題解決が必要
- 複数のメトリクスとログの相関分析が重要

#### システム管理者
- 複数のメトリクスとログを相関させて根本原因を特定
- オンコール対応の効率化が課題

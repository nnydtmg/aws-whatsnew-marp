# The AWS MCP Server is now generally available

**カテゴリ:** What's New
**公開日:** 2026-05-06T12:00:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-mcp-server/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-mcp-server/)

---

## 要約

AWS MCP Serverが一般提供開始となり、AIコーディングエージェントがIAMベースのガードレールと監査ログを通じてAWSサービスに安全にアクセスできるようになりました。このアップデートは、セキュリティと可視性を維持しながらエージェントベースの自動化を実装したい組織に適しており、追加料金なしで利用可能です。

## 詳細

**新機能について:**
- AWS MCP Serverが一般提供開始となり、AIコーディングエージェントがModel Context Protocol（MCP）を通じてAWSサービスに安全で監査可能なアクセスができるようになりました。

- IAMベースのガードレール、Amazon CloudWatchメトリクス、AWS CloudTrailログを通じて、可視性と制御を維持しながらエージェントがAWSと対話できます。

- エージェントが単一のツールを通じてあらゆるAWS APIを呼び出せるようになり、ファイルアップロードや長時間実行操作にも対応しています。

- サンドボックス化されたPythonスクリプト実行により、ローカルファイルシステムやシェルツールへのアクセスなしに複数ステップの操作が可能になりました。

- エージェントスキルにより、キュレーションされたガイダンスをオンデマンドで検出・読み込みでき、コンテキストウィンドウ使用量を低く保ちながら複雑なタスク手順を提供します。

- ドキュメント検索とスキル検出がAWS認証情報を不要にし、導入の障壁を低減しています。

- 追加料金なしで利用可能であり、エージェントが使用するAWSリソースのみの料金です。

- US East（N. Virginia）およびEurope（Frankfurt）で利用可能です。

**このアップデートが適している対象:**
- AWSサービスと連携するAIコーディングエージェントを構築・運用する組織です。

- セキュリティと監査可能性を維持しながら、エージェントに自動化タスクを実行させたい企業です。

- 複雑なマルチステップ操作を効率的に実行する必要があるAWS利用者です。

- エージェント開発の導入障壁を低減したいデベロッパーチームです。

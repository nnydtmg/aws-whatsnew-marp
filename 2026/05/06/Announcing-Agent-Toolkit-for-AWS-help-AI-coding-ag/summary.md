# Announcing Agent Toolkit for AWS — help AI coding agents build effectively on AWS

**カテゴリ:** What's New
**公開日:** 2026-05-06
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/05/agent-toolkit/](https://aws.amazon.com/about-aws/whats-new/2026/05/agent-toolkit/)

---

## 要約

AWS用Agent Toolkitは、AIコーディングエージェントがAWS上で複雑なマルチサービスワークフローを効率的に実行するための本番環境対応ツールスイートであり、アプリケーション開発者、データアナリスト、AIエンジニアなど、様々な開発者向けの包括的なソリューションを提供しています。

## 詳細

### 新機能について

- AWS用Agent Toolkitは、AIコーディングエージェントがAWS上で効果的に構築するための本番環境対応ツールスイートです。
- 40以上のエージェントスキルが提供されており、インフラストラクチャコード、ストレージ、分析、サーバーレス、コンテナ、AIサービスなどの領域をカバーしています。
- 完全管理型のMCPサーバーが一般提供されており、IAMベースのガードレール、CloudWatchおよびCloudTrailの可視性、マルチステップ操作のサンドボックス実行が提供されます。
- 3つのエージェントプラグイン（AWS Core、AWS Data Analytics、AWS Agents）が利用可能です。
- 追加料金なしで利用でき、使用したAWSリソースの料金のみが発生します。

### このアップデートが適している対象者

- **AIコーディングエージェントを使用してAWS上で構築する開発者**
  - 複雑なマルチサービスワークフローの実装が必要
  - AIエージェント技術を活用したい

- **フルスタックアプリケーション開発者**
  - フルスタックアプリケーションを構築・管理する必要がある
  - インフラストラクチャ自動化が必要

- **データアナリスト・ビジネスインテリジェンスエンジニア**
  - データパイプラインの作成やデータクエリを行う
  - SQLベースの分析作業を効率化したい

- **AIエンジニア**
  - Amazon Bedrockを使用して本番環境対応のエージェントを構築
  - エンタープライズグレードのセキュリティが必要

## 主な利点

### 開発効率の向上

- 40以上のスキルでAWSサービスへの即座なアクセス
- インフラストラクチャコードの自動生成
- AWSサービス詳細知識の不要化

### セキュリティと信頼性

- IAMベースの細粒度アクセス制御
- CloudWatchおよびCloudTrailによる完全な監視
- 本番環境対応のエンタープライズセキュリティ

### コスト最適化

- Agent Toolkit利用は追加料金なし
- 使用したAWSリソースの料金のみ発生
- リソース使用の最適化が可能

## 関連技術・背景

- **Amazon Bedrock** : マネージド型生成AIサービス
- **MCP (Model Context Protocol)** : AI モデルとツール統合の標準
- **IAM (Identity and Access Management)** : AWS アクセス管理
- **CloudWatch / CloudTrail** : AWS 監視・ロギングサービス

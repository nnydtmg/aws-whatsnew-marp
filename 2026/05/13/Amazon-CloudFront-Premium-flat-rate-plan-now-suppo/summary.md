# Amazon CloudFront Premium フラットレート プランで設定可能な使用量許容量に対応

Amazon CloudFront Premium flat-rate plan now supports configurable usage allowances

**カテゴリ:** What's New
**公開日:** 2026-05-12T16:00:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/cloudfront-configurable-premium-flat-rate-plans/)

このページでは、AWS What's Newで発表された「Amazon CloudFront Premium flat-rate plan now supports configurable usage allowances」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

Amazon CloudFrontのPremiumフラットレートプランが、500万～60億リクエスト、50TB～600TBの範囲で設定可能な使用量許容量に対応し、エンタープライズおよび中堅企業がアプリケーションの成長に応じて柔軟にスケーリングできるようになりました。CloudFrontコンソールから即座に料金確認でき、契約期間の制限なくいつでも使用量レベルを変更できます。

## このアップデートで何が変わったか

- Amazon CloudFrontのPremiumフラットレートプランが、設定可能な使用量許容量に対応するようになりました。
- 従来は単一の使用量許容量のみでしたが、現在は月間500万リクエストから60億リクエスト、50TBから600TBまでの範囲で自動調整が可能になりました。
- CloudFrontコンソールから使用量レベルを選択でき、月額固定料金が即座に表示されます。
- 使用量レベルはいつでも変更可能であり、契約期間の制限がありません。
- すべてのPremiumプラン機能が各使用量レベルで利用可能です。
- フラットレートプランには、コンテンツ配信、AWS WAF、DDoS保護、ボット管理、Amazon Route 53 DNS、Amazon CloudWatch Logsの取り込み、サーバーレスエッジコンピュート、Amazon S3ストレージクレジットが含まれており、追加料金は発生しません。
- このアップデートは、従来はフラットレートプランの対象外だったエンタープライズおよび中堅企業に適しています。
- アプリケーションの成長に伴い、Premiumプラン内でスケーリングできるようになりました。

## 対象ユーザー

- エンタープライズ企業
- 中堅企業
- 成長期のスタートアップ
- SaaS企業
- eコマースサイト
- ストリーミングサービス
- API配信事業者

## 詳細

### 使用量許容量の範囲

- **月間リクエスト数**: 500万～60億
- **月間データ転送**: 50TB～600TB

### CloudFrontコンソールでの操作

- 使用量レベルを選択するだけで月額固定料金が即座に表示
- 別途AWSサポートに連絡する必要なし
- CloudFrontコンソール内で完結

### 変更のシンプルさ

- いつでも使用量レベルを変更可能
- 契約期間の制限なし
- 長期コミットメント不要

### 包含サービス

- コンテンツ配信
- AWS WAF
- DDoS保護
- ボット管理
- Amazon Route 53 DNS
- Amazon CloudWatch Logs 取り込み
- サーバーレスエッジコンピュート
- Amazon S3 ストレージクレジット
- **追加料金なし**

## 効果・メリット

### 1. コスト予測可能性
- 月額固定料金で予算計画が容易
- オーバーチャージのリスク排除

### 2. スケーリング柔軟性
- エンタープライズ・中堅企業が容易に採用可能
- 成長段階に応じた段階的スケールアップ

### 3. 運用負担軽減
- CloudFrontコンソールから自動調整
- AWSサポート連絡が不要

### 4. セキュリティ統合
- WAF、DDoS保護が月額固定料金に含まれる
- 追加費用なくセキュリティ強化

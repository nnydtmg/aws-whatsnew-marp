AWS Security Agent now supports full repository code reviews

## 要約

AWS Security Agent の全リポジトリコードレビュー機能により、従来のパターンマッチング手法では検出できないシステム的な脆弱性を深い文脈認識型の分析で発見し、具体的な修復コードを自動生成することが可能になりました。本機能は既存顧客向けにプレビュー期間中は無料で提供されており、セキュリティチームが脆弱性の特定と修復を迅速に行うために特に有効です。

## 詳細

- AWS Security Agent は、2026年5月12日に全リポジトリコードレビュー機能をリリースしたことを発表いたしました。

- この新機能は、コードベース全体に対して深い文脈認識型のセキュリティ分析を実行するものです。

- 従来の静的分析ツールとは異なり、既知の脆弱性パターンとの照合ではなく、アプリケーションのアーキテクチャ、信頼境界、データフローを分析して、パターンマッチングツールでは検出できないシステム的な脆弱性を発見いたします。

- 脆弱性が検出された場合、スキャナーは特定のファイルと行番号に紐付けられた具体的な修復コードを生成いたします。

- この機能は、既存のAWS Security Agent顧客向けにプレビュー期間中は追加料金なしでご利用いただけます。

- 全リポジトリコードレビューは、AWS Security Agentが利用可能なすべてのAWSリージョンで提供されております。

- このアップデートは、セキュリティチームがコードベースを強化し、脆弱性をより迅速に特定・修復する必要がある組織に適しております。

- 開発チームやセキュリティ運用チームが、AIドリブンのセキュリティ分析を活用して、システム的な脆弱性に対応したい場合に特に有効です。

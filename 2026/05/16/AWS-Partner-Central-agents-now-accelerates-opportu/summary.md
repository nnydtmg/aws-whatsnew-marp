# AWS Partner Central agentsが営業機会の作成を加速

AWS Partner Central agents now accelerates opportunity creation

**カテゴリ:** What's New
**公開日:** 2026-05-15T18:41:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-partner-central-agents-oppo)

---

## 要約

AWS Partner Central agentsは、自然言語会話を通じて営業機会の作成を加速させ、パートナーがデータ入力に費やす時間を削減して営業活動に集中できるようにする新機能です。このアップデートは、営業チームがパイプラインの衛生管理を改善し、営業サイクルを短縮するのに適しています。

---

## このアップデートで何が変わったか

### 主要な変更点

1. **自然言語での営業機会作成**
   - パートナーが短い会話で営業機会を作成できるように進化
   - 複数ステップの面倒なフォーム入力が不要に
   - 直感的で素早い操作が可能

2. **情報自動抽出・充実化**
   - 会議メモ、提案書、通話記録（PDF、DOCX、Excel、TXT形式）をアップロード
   - 既存の営業機会を複製して活用
   - エージェントが情報を自動抽出し、顧客詳細を充実化

3. **エージェントによる改善提案**
   - 不足しているコンテキストの追加
   - フィールド値の修正
   - ビジネス問題ステートメントの強化
   - より高品質な営業機会を実現

4. **複数のアクセス方法**
   - AWS Console経由: Amazon Q chatを通じた直感的操作
   - プログラマティック: Model Context Protocol (MCP)を使用した既存ツールとの統合

5. **グローバル対応**
   - すべての商用AWSリージョンで利用可能
   - グローバルな営業組織の効率化を実現

---

## 効果・メリット

### パートナー営業チームへのメリット

- **時間効率化**: データ入力に費やす時間を大幅削減
- **営業活動への集中**: 営業チームが実際の営業活動に専念可能に
- **品質向上**: エージェントによる改善提案で高品質な営業機会を作成
- **営業ツール統合**: 既存の営業ツールから直接利用可能

### 営業組織への効果

- **パイプライン衛生管理**: 営業機会の質が向上、精度が改善
- **営業サイクル短縮**: より早い案件進捗と迅速な顧客対応
- **プロセス標準化**: AIエージェントによる統一されたプロセス
- **データ駆動型営業**: より正確なデータに基づいた営業活動

## ユースケース

### 営業チーム

1. **会議記録の自動化**
   - 顧客会議のメモをアップロード
   - 自動で営業機会を作成
   - 手動入力不要で即座に記録

2. **提案書からの情報抽出**
   - 提案書をアップロード
   - 提案内容から自動で営業機会を生成
   - 提案フォローアップを効率化

3. **通話記録の活用**
   - 通話記録をアップロード
   - 通話内容から営業機会の詳細を自動抽出
   - フォローアップを迅速化

4. **テンプレート利用**
   - 既存の成功事例を複製
   - 同様の機会を素早く作成
   - 営業チーム全体の生産性向上

### パートナー企業全体

- グローバル営業組織の効率化
- 営業プロセスの統一と標準化
- データ品質の向上による意思決定の改善
- AI活用による競争力向上

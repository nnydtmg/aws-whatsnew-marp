# Amazon Bedrock AgentCore Runtime adds WebRTC support for real-time bidirectional streaming

**カテゴリ:** What's New
**公開日:** 2026-03-20T21:54:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-bedrock-webrtc/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-bedrock-webrtc/)

---

## 要約

Amazon Bedrock AgentCore Runtime に WebRTC サポートが追加され、ブラウザおよびモバイルアプリケーション向けの低遅延リアルタイム音声・ビデオエージェント開発が可能になりました。複数の TURN リレーオプションと 14 の AWS リージョンでの利用により、開発者は柔軟で拡張性の高い音声エージェント実装を構築できます。

## 詳細

- Amazon Bedrock AgentCore Runtime が WebRTC サポートを追加し、リアルタイム双方向ストリーミング機能が拡張されました。

- WebRTC は UDP ベースのピア・ツー・ピア通信により、低遅延のオーディオおよびビデオ双方向ストリーミングを実現します。

- 既存の WebSocket プロトコルに加えて、WebRTC は 2 番目の双方向ストリーミングプロトコルとして提供されます。

- WebRTC は Amazon Kinesis Video Streams の管理型 TURN、サードパーティプロバイダー、またはセルフホスト型 TURN インフラストラクチャなど、複数の TURN リレーオプションをサポートしています。

- 本機能は 14 の AWS リージョンで利用可能です。

- ブラウザおよびモバイルアプリケーション向けの音声エージェント開発に最適です。

- 低遅延が重要なリアルタイムメディア配信シナリオに適しています。

- Amazon Nova Sonic 音声エージェント、Pipecat 音声エージェント、LiveKit 音声エージェント、Strands Agents SDK 音声エージェントなどのデプロイ可能な例が提供されています。

# Accelerate serverless application development with new SAM Kiro power

**カテゴリ:** What's New
**公開日:** 2026-03-13
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-sam-kiro-power/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-sam-kiro-power/)

---

## 要約

AWS SAM Kiro powerは、Kiroの環境でAIエージェントの支援を受けながらサーバーレスアプリケーションの構築からデプロイまでを効率的に行える新機能です。このアップデートは、開発生産性の向上とベストプラクティスの自動適用を求める開発者やアーキテクトに特に有用です。

## 詳細

- AWS Serverless Application Model (SAM) Kiro powerは、Kiroにおけるエージェント型AI開発にサーバーレスアプリケーション開発の専門知識をもたらす新機能です。
- ローカル環境でAIエージェントの支援を受けながら、サーバーレスアプリケーションの構築、デプロイ、管理が可能になります。
- SAM プロジェクトの初期化、AWSへのアプリケーションのビルドおよびデプロイ、Lambda関数のローカルテストが動的にサポートされます。
- Amazon EventBridge、Amazon MSK、Amazon Kinesis、Amazon DynamoDB Streams、Amazon SQSなどのイベント駆動パターンに対応しています。
- IAMポリシーのセキュリティベストプラクティスがカバーされており、デフォルトでAWS Lambda PowertoolsとSAMリソースの使用が推奨されます。
- この機能は、静的ウェブサイト、イベント駆動型マイクロサービス、フルスタックアプリケーションなど、様々なユースケースに対応しています。
- Kiro IDE内から1クリックでインストール可能です。
- このアップデートは、サーバーレスアプリケーション開発を行う開発者やソリューションアーキテクト、AIエージェント支援による開発効率化を求める組織に適しています。

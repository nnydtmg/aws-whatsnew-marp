# AWS CDK Mixins is now generally available

**カテゴリ:** What's New
**公開日:** 2026-03-12T16:15:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-cdk-mixins/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-cdk-mixins/)

---

## 要約

AWS CDK Mixinsは、既存のインフラストラクチャコードを再構築することなく、あらゆるAWSコンストラクトに再利用可能な抽象化を追加できる新機能として一般提供が開始されました。このアップデートは、セキュリティ、コンプライアンス、運用ポリシーを効率的に適用する必要があるエンタープライズチームや、新しいAWS機能への早期アクセスと高度な抽象化の両方を求める開発チームに特に有益です。

## 詳細

- AWS CDK Mixinsは、AWS Cloud Development Kit（CDK）の新機能として一般提供が開始されました。

- CDK Mixinsは、L1、L2、またはカスタムコンストラクトなど、あらゆるAWSコンストラクトに対して、既存のインフラストラクチャコードを再構築することなく、再利用可能で構成可能な抽象化を追加することができます。

- .with()構文を使用して、自動削除、バケット暗号化、バージョニング、パブリックアクセスのブロックなどの機能をコンストラクトに直接適用することができます。

- 複数のMixinsをカスタムL2コンストラクトに組み合わせたり、スコープ全体にコンプライアンスポリシーを適用したりすることが可能です。

- Mixins.of()を使用することで、高度なリソースタイプまたはパスパターンフィルタリングを実行できます。

- エンタープライズチームは、新しいAWS機能への初日アクセスを維持しながら、インフラストラクチャ全体にセキュリティとコンプライアンスポリシーを適用できます。

- CDK Mixinsはaws-cdk-libパッケージを通じて利用可能であり、AWS CloudFormationがサポートされているすべてのAWSリージョンで利用できます。

- このアップデートは、セキュリティ、コンプライアンス、運用要件を満たすために大幅な作業が必要だった開発チームに特に有益です。

- カスタムコンストラクトライブラリの保守を簡素化する必要があるエンタープライズ組織にとって有用です。

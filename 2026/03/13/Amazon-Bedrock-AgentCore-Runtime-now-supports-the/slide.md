# Amazon Bedrock AgentCore Runtime
AG-UIプロトコルに対応

**発表日: 2026年3月13日**

---

## 概要

### AG-UI プロトコル対応のポイント

- **リアルタイム応答**: ユーザー向けアプリケーションでリアルタイムで応答性の高いエージェント体験を実現
- **インフラストラクチャ自動化**: 認証、セッション分離、スケーリングを自動管理
- **開発に専念**: インタラクティブなフロントエンド開発に注力可能

---

## 前提・背景

### AI エージェント開発の課題

- 複雑なインフラ管理が必要（認証、スケーリング、セッション管理）
- UI との統合が困難で、ユーザー体験の構築に時間がかかる
- リアルタイム通信の実装が複雑

### Amazon Bedrock AgentCore Runtime の進化

- **MCP** (Model Context Protocol): ツール・リソース管理
- **A2A** (Agent-to-Agent): エージェント間通信
- **AG-UI** (Agent-User Interface): エージェント・UI 間通信 ← **今回のリリース**

---

## 主な機能・特徴

### AG-UI プロトコルの概要

- **オープン標準**: イベントベースのオープンプロトコル
- **標準化通信**: AI エージェントとユーザーインターフェースの通信を標準化
- **MCP と A2A 連携**: 既存プロトコルとの相補的な動作

---

## 主な機能・特徴（続き）

### リアルタイムストリーミング機能

- **テキストチャンク**: 推論途中のテキストを段階的に配信
- **推論ステップ**: エージェントの推論プロセスをリアルタイム表示
- **ツール結果**: 外部ツール実行結果をフロントエンドにストリーミング

---

## 主な機能・特徴（続き）

### UI 統合機能

- **リアルタイム状態同期**
  - プログレスバーの動的更新
  - ダッシュボード要素のライブ更新

- **構造化ツール呼び出し表示**
  - エージェントアクションの透過的レンダリング
  - ユーザーによる理解促進

---

## 主な機能・特徴（続き）

### トランスポート & 地域対応

- **双方向通信**
  - **SSE** (Server-Sent Events): サーバーからクライアントへの単方向ストリーミング
  - **WebSocket**: 双方向リアルタイム通信

- **14 AWS リージョン対応**
  - 米国（バージニア、オハイオ、オレゴン）
  - アジア太平洋（ムンバイ、ソウル、シンガポール、シドニー、東京）
  - カナダ、ヨーロッパ（フランクフルト、アイルランド、ロンドン、パリ、ストックホルム）

---

## インフラストラクチャ管理の自動化

### AgentCore Runtime が提供

- **認証管理**: OAuth 2.0、SigV4 など複数の認証方式をサポート
- **セッション分離**: 各ユーザーセッションを分離・隔離
- **自動スケーリング**: ワークロード対応の動的スケーリング

### 開発者のメリット

- インフラ構築コストの削減
- セキュリティベストプラクティスの自動適用
- 本番環境での信頼性向上

## 効果・メリット

### 開発効率向上

- **実装期間の短縮**: インフラ管理から解放、UI 開発に注力
- **スケーラビリティ**: 自動スケーリングで大規模展開可能
- **保守性向上**: 管理負荷の軽減

### ユーザー体験改善

- **リアルタイム応答**: エージェントの動作をリアルタイム表示
- **透明性**: 推論ステップ、ツール呼び出しを可視化
- **インタラクティブ性**: 双方向通信により高度なインタラクション実現

## ユースケース

### 対象となる開発者・組織

- **エージェント技術を活用したアプリケーション開発**: AI エージェント統合が必要な企業
- **リアルタイム応答性が求められるシステム**
  - 顧客サポートチャットボット
  - 分析・レポート生成ツール
  - インタラクティブなデータ分析アシスタント

### 具体的な活用例

## まとめ

### Amazon Bedrock AgentCore Runtime の全体像

- **MCP**: エージェントのツール・リソース管理
- **A2A**: エージェント間の協調・通信
- **AG-UI**: **エージェント・ユーザー間のインタラクション** ← 今回の新機能

### ビジネスへの影響

- 高度なエージェント体験の構築が容易化
- デプロイ・運用コストの低減
- 市場投入時間（TTM）の短縮

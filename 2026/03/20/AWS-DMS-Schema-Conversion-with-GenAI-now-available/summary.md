# AWS DMS Schema Conversion with GenAI now available in 10 additional regions

**カテゴリ:** What's New
**公開日:** 2026-03-20
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-dms-schema-conversion-with-genai/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-dms-schema-conversion-with-genai/)

---

## 要約

AWS DMS Schema Conversion with GenAIが9つの追加地域で利用可能になり、Amazon Bedrockの生成AIモデルを活用してデータベーススキーマとコード変換を自動化することで、マイグレーション作業の効率化とデータレジデンシー要件への対応が実現されました。このサービスは追加料金なしで提供され、データベース管理者とマイグレーション専門家の負担を大幅に軽減します。

## 詳細

AWS DMS Schema Conversion with GenAIが、アジア太平洋地域（東京、大阪、シドニー）、ヨーロッパ地域（アイルランド、ロンドン、ストックホルム、パリ）、カナダ（中部）、米国東部（オハイオ）の9つの追加AWS地域で利用可能になりました。このサービスはAmazon Bedrockの基盤モデル（Claude 3.5 Sonnet v2、Claude 3.7 Sonnet、Claude Sonnet 4を含む）を活用して、データベーススキーマとコード変換を自動化します。Oracle、SQL Server、MySQL、PostgreSQL、SybaseからAmazon Aurora PostgreSQL互換版およびAmazon RDS for PostgreSQLへの自動変換に対応しています。手動作業を大幅に削減し、マイグレーションプロジェクトのタイムラインを加速させることができます。ローカルでマイグレーションワークロードを処理することで、レイテンシを削減し、データレジデンシー要件をサポートします。DMS Schema Conversionは追加料金なしで利用でき、AWS Management ConsoleまたはAWS Command Line Interface（CLI）からアクセスできます。このアップデートは、データベース管理者とマイグレーション専門家が戦略的な最新化活動に集中できるようにします。

# Announcing AWS Partner Central agents to accelerate co-sell

**カテゴリ:** What's New
**公開日:** 2026-03-16T17:04:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-partner-central-agents-accelerate-co-sell/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-partner-central-agents-accelerate-co-sell/)

---

## 要約

AWS Partner Central agentsは、Amazon Bedrock AgentCoreを基盤とした新しいAI駆動型機能であり、パートナー企業の営業チームが営業サイクルを短縮し、資金調達プロセスを簡素化することを支援します。このアップデートは、営業効率を向上させ、データ入力作業を削減したいAWSパートナー企業の営業チームに特に有益です。

## 詳細

### 主な機能

1. **AI駆動型エージェント**
   - Amazon Bedrock AgentCoreを基盤とした新機能
   - パートナー企業の共同販売を加速
   - 営業サイクルを短縮することを目的とした設計

2. **営業インサイト機能**
   - パイプラインの洞察を提供
   - カスタマイズされた営業戦略を提案
   - 次のステップの推奨事項をオンデマンドで提供

3. **自動データ入力機能**
   - 会議の記録、メモ、メールを共有
   - 自動的にフィールドを入力し案件を進行
   - データ入力作業を削減
   - 営業チームが販売に集中できるよう支援

4. **資金調達支援機能**
   - 機会レベルでの資金調達の推奨を実施
   - 適格性ギャップを強調
   - 事前入力された資金要求を作成
   - 利用可能な資金をより迅速に確保可能

### 利用方法

- **AWS Console**: 直接AWS Consoleで利用可能
- **プログラマティックアクセス**: Model Context Protocol (MCP)を通じてプログラムで利用可能
- **CRM統合**: CRMシステム内から直接エージェント機能にアクセス可能

### 提供地域と可用性

- すべての商用AWSリージョンで本日から利用可能
- 2026年3月16日より一般提供開始

### ビジネスメリット

- **営業効率の向上**: 営業チームが販売活動に集中できる環境を実現
- **営業サイクルの短縮**: 自動化による効率的なプロセス実行
- **資金調達の加速**: 申請から承認までのプロセスを簡素化
- **データ入力時間の削減**: 手作業を大幅に削減し生産性を向上

### ユースケース

1. **スタートアップ向け**: 限られたリソースで営業効率を最大化
2. **エンタープライズパートナー向け**: 大規模案件管理と戦略最適化
3. **資金調達最大化**: AWS資金プログラムの効率的な活用

### 次のステップ

1. AWS Consoleでアクセスして評価開始
2. 小規模パイロットで効果測定
3. 営業チーム全体への段階的展開
4. プロセス最適化と継続的改善

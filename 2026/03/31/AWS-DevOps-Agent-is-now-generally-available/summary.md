# AWS DevOps Agent is now generally available

**カテゴリ:** What's New
**公開日:** 2026-03-31
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-devops-agent-generally-available/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-devops-agent-generally-available/)

---

## 要約

AWS DevOps Agentが一般提供開始となり、AWS、マルチクラウド、オンプレミス環境全体でインシデント解決と予防的な運用改善を行う機能が拡張されました。このアップデートは、運用チームの効率化とMTTRの短縮を求める企業、特にAWS Support顧客にとって有用です。

## 詳細

AWS DevOps Agentが2026年3月31日に一般提供開始となりました。AWS、マルチクラウド、オンプレミス環境全体にわたるインシデント解決と予防的なインシデント防止を行う常時利用可能な運用チームメンバーです。Azureおよびオンプレミス環境のアプリケーション調査機能が新たに追加されました。カスタムエージェントスキルを追加してCapabilityを拡張できるようになりました。運用インサイト向けのカスタムチャートとレポート作成機能が実装されました。アプリケーションの関係性を学習し、可観測性ツール、ランブック、コードリポジトリ、CI/CDパイプラインと連携して動作します。平均復旧時間(MTTR)を数時間から数分に短縮できます。過去のインシデントパターンを分析して、将来の障害を防ぐための実行可能な推奨事項を提供します。AWS Support顧客は月次のDevOps Agentクレジットを受け取ります(Unified Operations 100%、Enterprise Support 75%、Business Support+ 30%)。プレビュー顧客は新機能へのシームレスなアクセスのため、移行ドキュメントを確認する必要があります。

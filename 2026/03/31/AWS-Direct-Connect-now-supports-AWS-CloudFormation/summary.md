# AWS Direct Connect now supports AWS CloudFormation

**カテゴリ:** What's New
**公開日:** 2026-03-31
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-direct-connect-supports-aws-cloudformation/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-direct-connect-supports-aws-cloudformation/)

---

## 要約

AWS Direct Connect が CloudFormation に対応し、接続、仮想インターフェース、ゲートウェイなどの Direct Connect リソースを Infrastructure-as-Code で定義・管理できるようになりました。これにより、全世界のすべての AWS リージョンで、バージョン管理とドリフト検出を備えた反復可能なデプロイメントが実現します。

## 詳細

AWS Direct Connect が AWS CloudFormation に対応しました。これにより、Infrastructure-as-Code を使用した Direct Connect リソースのプロビジョニングが可能になります。CloudFormation テンプレートを使用して、Direct Connect トポロジ全体を他の AWS リソースと一緒に定義できるようになりました。バージョン管理、反復可能なデプロイ、ドリフト検出、および変更管理機能が利用可能になります。接続、仮想インターフェース、Direct Connect ゲートウェイ、Link Aggregation Group（LAG）、および BGP ピアリング設定の作成と管理を自動化できます。この機能は、全世界のすべての AWS リージョンで利用可能です。このアップデートは、インフラストラクチャをコードで管理したい企業や、複数の Direct Connect リソースを一元管理したい組織に適しています。自動化とバージョン管理を重視する DevOps チームやクラウドアーキテクトにとって有益です。

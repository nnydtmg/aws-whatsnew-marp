# AWS announces End User Messaging Notify

**公開日:** 2026-03-31T17:24:00

## 要約

AWS End User Messaging Notifyは、開発者がAWSが所有する電話番号を活用して数分以内にOTP送信を開始できる新しいサービスであり、従来の数週間から数ヶ月の設定期間を大幅に短縮します。このサービスは、顧客認証を迅速に実装したい企業や開発者にとって、組み込みの詐欺対策と支出制限機能を備えた理想的なソリューションです。

## 詳細

### サービス概要

- AWS End User Messaging Notifyは、ワンタイムパスコード（OTP）を数分以内に送信開始できる新しいサービスです。

- 従来は電話番号取得とキャリア登録に数週間から数ヶ月を要していましたが、このサービスではAWSが所有する電話番号と送信者IDを利用できます。

### 主な機能

- SMS、音声、またはその両方を有効にして、200以上の国にOTPメッセージを送信することができます。

- ブランド名、コード形式、コードの有効期限をカスタマイズ可能です。

- すべてのAPI呼び出しにAWS End User Messaging SMS Protectによる組み込みのSMS詐欺対策が追加費用なしで含まれています。

- 支出制限機能により、設定した閾値に達した場合に配信を一時停止することで、追加の保護層が提供されます。

### 提供範囲

- このサービスはAWS End User Messagingが利用可能なすべてのAWSリージョンで利用できます。

- 開発者は迅速にOTP認証を実装でき、セキュリティ機能も備わっています。

## メリット

### 導入の迅速化
- 設定時間が数週間～数ヶ月から数分に短縮
- すぐに本番環境での利用が可能

### セキュリティの強化
- 組み込みの詐欺対策機能
- 支出制限による保護
- リアルタイム監視機能

### グローバル対応
- 200以上の国で利用可能
- マルチチャネル（SMS、音声）対応
- 多言語サポート

### コスト効率
- AWS所有番号使用による管理コスト削減
- 個別の電話番号取得・保有コストがかからない
- 詐欺対策が標準装備で追加費用なし

# AWS Builder ID now supports Sign in with GitHub and Amazon

**カテゴリ:** What's New
**公開日:** 2026-03-10
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-builder-id-sign-in-github-amazon/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-builder-id-sign-in-github-amazon/)

---

## 要約

AWS Builder IDがGitHubおよびAmazonアカウントでのサインイン機能に対応し、既存のGoogle・Appleのオプションに加えて、ユーザーの認証選択肢が拡大されました。この更新により、開発者やビルダーはパスワード管理の複雑性を軽減しながら、AWS Builder Center、AWS Training and Certification、Kiroなどのサービスへシームレスにアクセスできるようになります。

## 詳細

AWS Builder IDが、GitHubとAmazonアカウントでのサインイン機能に対応いたしました。既存のGoogleとAppleのソーシャルログイン機能に加えて、新たにGitHubとAmazonのサインイン選択肢が拡大されました。AWS Builder Center、AWS Training and Certification、およびKiroなどのAWSアプリケーションへのアクセスが、GitHubまたはAmazonの認証情報を使用して可能になりました。開発者とビルダーは、別途のAWS認証情報を管理する必要がなくなり、既存のGitHubまたはAmazonアカウントをセキュアなゲートウェイとして利用できるようになりました。パスワード管理の複雑性が軽減され、パスワード忘却の問題が削減され、新規ユーザー登録と既存ユーザーのサインインの両方においてシームレスな体験が提供されます。開発リソースへのアクセス、認定資格プログラムへの登録、およびKiroを使用したアプリケーション開発など、複数のAWSサービスへのアクセスが統一された認証で可能になります。

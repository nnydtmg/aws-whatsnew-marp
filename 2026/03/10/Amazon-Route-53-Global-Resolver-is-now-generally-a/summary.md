# Amazon Route 53 Global Resolver is now generally available

**カテゴリ:** What's New
**公開日:** 2026-03-09T22:04:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-route-53-global-resolver/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-route-53-global-resolver/)

---

## 要約

Amazon Route 53 Global Resolverは、30のAWSリージョンで一般提供開始された、インターネットに接続可能なエニーキャストDNSリゾルバーサービスです。このサービスは、認可されたクライアントに対して安全で信頼性の高いDNS解決、高度なDNS脅威フィルタリング、および集中化されたクエリログ機能を提供しており、組織全体のDNSセキュリティとガバナンスの強化に適しています。

## 詳細

- Amazon Route 53 Global Resolverが2026年3月9日に一般提供開始されました。
- インターネットに接続可能なエニーキャストDNSリゾルバーであり、認可されたクライアントに対して安全で信頼性の高いDNS解決を提供します。
- 30のAWSリージョンで利用可能であり、IPv4およびIPv6のDNSクエリトラフィックに対応しています。
- 組織内の認可されたクライアントは、任意の場所からパブリックインターネットドメインおよびRoute 53プライベートホストゾーンに関連するプライベートドメインのエニーキャストDNS解決を利用できます。
- DNS脅威フィルタリング機能により、悪意のあるドメイン、不適切なコンテンツを含むドメイン、DNSトンネリングやDomain Generation Algorithms（DGA）などの高度なDNS脅威に関連するドメインをブロックできます。
- 一般提供開始に伴い、Dictionary DGA脅威に対する保護が追加されました。
- 集中化されたクエリログ機能が提供されます。
- 新規顧客は30日間の無料トライアルを利用できます.

# Amazon Quick Suite launches User Preferences for chat personalization

**カテゴリ:** What's New
**公開日:** 2026-03-09
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/user-preferences-in-quick/](https://aws.amazon.com/about-aws/whats-new/2026/03/user-preferences-in-quick/)

---

## 要約

Amazon Quick Suiteに新たに追加された「User Preferences」機能により、ユーザーはチャットパネルのレイアウト、デフォルトエージェント、個人情報などをカスタマイズでき、セッション間で設定が保持されるようになりました。この機能により、ユーザーは毎回の操作がより効率的でパーソナライズされた体験を得ることができます。

## 詳細

### 新機能の概要

- Amazon Quick Suiteに「User Preferences」という新機能が追加されました
- ユーザーは個人設定をカスタマイズし、セッション間で保持できるようになります
- 前回のセッション状態から自動的に再開できる機能を提供

### 主な機能

#### 1. チャットパネル レイアウト設定

- チャットパネルのレイアウトを展開状態または折りたたみ状態でデフォルト表示するようにカスタマイズ可能
- ユーザーの最後に使用した設定を自動的に記憶
- 次回アクセス時に前回の状態から再開できます

#### 2. エージェント選択

- デフォルトのチャットエージェントを選択可能
- My Assistantのデフォルト知識範囲を事前選択できます
- エージェント切り替えの手間を削減

#### 3. ユーザーコンテキスト

- ユーザーが自分の名前と仕事上の焦点領域を共有することで、Quickがより関連性の高いレスポンスを提供
- User Preferencesから直接メモリを表示・管理することができます

### これまでの課題

- 従来はセッション間でチャット設定、エージェント選択、個人コンテキストを保持する方法がありませんでした
- ユーザーは毎回設定を再入力する必要がありました
- セッション間での連続性が欠落していました

### 提供状況

- この機能はAmazon Quick Suiteが利用可能なすべてのAWSリージョンで利用できます
- 2026年3月9日より利用可能

### 対象ユーザー

- この更新は、セッション間での設定の永続化を必要とするユーザーに適しています
- 毎回の操作を効率化し、パーソナライズされた体験を求めるユーザーに最適です

## 効果とメリット

### ユーザー体験の向上

- セッション間での設定を自動保持
- 毎回のセットアップ時間を削減
- 一貫性のあるワークフロー環境を実現

### 生産性向上

- 効率的な作業継続
- パーソナライズされたAIレスポンス
- 操作習熟度の向上

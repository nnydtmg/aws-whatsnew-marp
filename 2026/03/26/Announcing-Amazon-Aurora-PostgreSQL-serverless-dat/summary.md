# Announcing Amazon Aurora PostgreSQL serverless database creation in seconds

**カテゴリ:** AWS Blog
**公開日:** 2026-03-25T20:37:11
**元記事:** [https://aws.amazon.com/blogs/aws/announcing-amazon-aurora-postgresql-serverless-database-creation-in-seconds/](https://aws.amazon.com/blogs/aws/announcing-amazon-aurora-postgresql-serverless-database-creation-in-seconds/)

---

## 要約

Amazon Aurora PostgreSQL サーバーレスデータベースをエクスプレス構成で数秒で作成できるようになり、IAM認証とインターネットアクセスゲートウェイがデフォルトで有効になっております。このアップデートは、迅速にアプリケーション開発を開始したい開発者やスタートアップ企業、ならびにプロトタイプ環境を素早く構築したいエンタープライズ組織に最適です。

## 詳細

- Amazon Aurora PostgreSQL サーバーレス データベースを数秒で作成できる新しいエクスプレス構成が一般提供されるようになりました。

- わずか2クリックで、本番環境対応のAurora PostgreSQLサーバーレスデータベースを数秒で起動できます。

- Amazon VPC ネットワークなしで作成され、インターネットアクセスゲートウェイが含まれており、VPNやAWS Direct Connectが不要です。

- IAM認証がデフォルトで有効になっており、パスワードレスデータベース認証が最初から利用可能です。

- AWS CloudShellまたはプログラミング言語を使用して、作成後すぐにAuroraデータベースをクエリできます。

- Vercel v0やKiro IDEなどの開発ツールとの統合により、自然言語を使用してアプリケーション構築を加速できます。

- このアップデートは、迅速にアプリケーション開発を開始したい開発者やスタートアップ企業に最適です。

- エンタープライズ組織でも、プロトタイプやPoC環境を数秒で構築したい場合に有用です。

- AWS Free Tierでも利用可能となり、追加コストなしで試験できます。

- すべてのAWSコマーシャルリージョンで本日から利用可能です。

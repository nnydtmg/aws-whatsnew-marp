# AWS Step Functions adds 28 new service integrations, including Amazon Bedrock AgentCore

**カテゴリ:** What's New
**公開日:** 2026-03-26
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-step-functions-sdk-integrations/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-step-functions-sdk-integrations/)

---

## 要約

AWS Step Functionsが28の新しいサービス統合と1,100以上のAPIアクションを追加し、Amazon Bedrock AgentCoreやAmazon S3 Vectorsなどの統合により、統合コードを記述することなくワークフローから直接AWSサービスを操作できるようになりました。このアップデートは、AIエージェント統合やドキュメント処理パイプラインが必要な分散アプリケーション開発者やエンタープライズユーザーに特に有用です。

## 詳細

AWS Step Functionsが28の新しいサービス統合を追加し、1,100以上の新しいAPIアクションが利用可能になりました。Amazon Bedrock AgentCoreとAmazon S3 Vectorsを含む新しいサービス統合により、統合コードを記述することなくワークフローから直接AWSサービスを操作できるようになりました。Amazon Bedrock AgentCore統合により、AIエージェントランタイムを組み込みリトライ機能付きで呼び出し、Map状態を使用して複数のエージェントを並列実行し、ワークフローステップとしてエージェントインフラストラクチャのプロビジョニングを自動化できます。Amazon S3 Vectorsサポートにより、AIアプリケーション用のナレッジベースを生成するドキュメント取り込みパイプラインの自動化が可能になりました。AWS Lambda耐久実行APIのサポートにより、Lambda耐久関数のべき等呼び出しのための実行名を渡し、ワークフローから直接耐久実行を管理できるようになりました。この機能拡張は、AWS Step Functionsが利用可能なすべてのAWSリージョンで一般提供されています。

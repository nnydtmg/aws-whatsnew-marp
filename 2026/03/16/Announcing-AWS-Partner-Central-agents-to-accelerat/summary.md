# Announcing AWS Partner Central agents to accelerate co-sell

**カテゴリ:** What's New
**公開日:** 2026-03-16
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-partner-central-agents-accelerate-co-sell/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-partner-central-agents-accelerate-co-sell/)

---

## 要約

AWS Partner Central agentsは、Amazon Bedrock AgentCoreを基盤とした新しいAI駆動型機能であり、パートナー企業の営業チームが営業サイクルを短縮し、資金調達プロセスを簡素化することを支援します。このアップデートは、営業効率を向上させ、データ入力作業を削減したいAWSパートナー企業の営業チームに特に有益です。

## 詳細

- AWS Partner Central agentsは、Amazon Bedrock AgentCoreを基盤とした新しいAI駆動型機能です。
- パートナー企業の共同販売を加速し、営業サイクルを短縮することを目的としています。
- パイプラインの洞察、カスタマイズされた営業戦略、次のステップの推奨事項をオンデマンドで提供します。
- 会議の記録、メモ、メールを共有することで、自動的にフィールドを入力し、案件を進めることができます。
- データ入力作業を削減し、営業チームが販売に集中できるようにします。
- 機会レベルでの資金調達の推奨と適格性ギャップの強調を行います。
- 事前入力された資金要求を作成し、利用可能な資金をより迅速に確保できます。
- AWS Consoleで直接利用するか、Model Context Protocol（MCP）を通じてプログラムで利用できます。
- CRMシステム内から直接エージェント機能にアクセスすることが可能です。
- すべての商用AWSリージョンで本日から利用可能です。

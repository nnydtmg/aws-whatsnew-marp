# Amazon Bedrock、OpenAI・Anthropic互換APIに最適化した再設計コンソールを提供開始

Amazon Bedrock launches a redesigned console optimized for OpenAI- and Anthropic-compatible APIs

**カテゴリ:** What's New
**公開日:** 2026-06-04
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-bedrock-redesigned-console-optimized-openai-anthropic-compatible-apis/)

## 要約

Amazon Bedrockの再設計されたコンソールにより、OpenAIおよびAnthropic互換APIを通じた基盤モデルの実験、比較、評価がシンプルに実現され、プロジェクト単位での統合ワークフローが提供されます。このアップデートは、生成AIアプリケーションの開発を迅速に進めたい開発者やエンタープライズ顧客に特に有用です。

## このアップデートで何が変わったか

- Amazon Bedrockのコンソールが再設計され、OpenAIおよびAnthropic互換APIに最適化されました。
- bedrock-mantleエンドポイントを通じて、OpenAI Responses API、OpenAI Chat Completions API、およびAnthropic Messages APIがサポートされるようになりました。
- モデルカタログの閲覧、機能やモダリティサポート、コンテキストウィンドウなどの比較が単一ビューで可能になりました。
- プロジェクト単位で評価と使用状況の分析が統合されたワークフローで実行できるようになりました。
- コード例やSDKスニペット、APIリファレンスが自動的にプロジェクトの設定値で事前入力されます。
- 開発者はコンソールからコードスニペットをコピーして、修正なしにアプリケーションで実行できます。
- 新しいコンソール体験は、US East、US West、Asia Pacific、Europe、South Americaなど複数のAWSリージョンで利用可能です。

## 対象ユーザー

このアップデートは、基盤モデルを使用したアプリケーション開発を迅速に進めたい開発者やデータサイエンティスト、ならびにOpenAIやAnthropic互換APIの利用を希望するエンタープライズ顧客に適しています。

# Amazon OpenSearch Service で AI支援の移行機能が利用可能に

Amazon OpenSearch Service now offers AI-assisted migrations

**カテゴリ:** What's New
**公開日:** 2026-06-23T03:30:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-opensearch-service-ai-migrations)

このページでは、AWS What's Newで発表された「Amazon OpenSearch Service now offers AI-assisted migrations」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

Amazon OpenSearch Service の Migration Assistant に AI 支援機能が追加され、自己管理型クラスターからの移行がより迅速かつ確実になりました。この更新は、複雑な移行プロセスを簡素化したいエンタープライズ組織や、Solr、Elasticsearch、OpenSearch を運用している企業に特に有益です。

## このアップデートで何が変わったか

- Amazon OpenSearch Service の Migration Assistant に AI 支援機能が追加されました。

- 本機能は Amazon OpenSearch Service が利用可能なすべての商用 AWS リージョンおよび AWS GovCloud（US）リージョンで利用可能です。

## 詳細

- Amazon OpenSearch Service の Migration Assistant に AI 支援機能が追加されました。

- 自己管理型の Apache Solr、Elasticsearch、または OpenSearch デプロイメントを OpenSearch Serverless またはマネージドクラスターへ移行する際に、AI ツール（Kiro、Claude Code など）を使用して計画、インフラストラクチャのデプロイ、履歴データおよびライブトラフィックの移行を実行できるようになりました。

- AI 支援ワークフローにより、データ移行の構造化、実行、検証をより迅速かつ確実に行うことができます。

- Solr に対するライブトラフィックキャプチャおよびリプレイ機能がサポートされるようになりました。

- Migration Assistant は OpenSearch Serverless とマネージドクラスターへの移行に対応しており、複数の Solr、Elasticsearch、OpenSearch バージョンからの移行をサポートしています。

- 本機能は Amazon OpenSearch Service が利用可能なすべての商用 AWS リージョンおよび AWS GovCloud（US）リージョンで利用可能です。

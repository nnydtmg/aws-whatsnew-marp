# AWS FinOps Agent がプレビューで利用可能に

AWS FinOps Agent is now available in preview

**カテゴリ:** What's New
**公開日:** 2026-06-09T15:28:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-finops-agent-preview/)

このページでは、AWS What's Newで発表された「AWS FinOps Agent is now available in preview」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

AWS FinOps Agent は、FinOps 実践者とエンジニアリングチームがコスト管理と最適化を効率的に行うための AI エージェントサービスであり、コスト分析、異常検知、自動調査、ワークフロー自動化などの機能を備えています。このサービスは、クラウドコストの可視化と最適化を求める組織にとって有益なツールとなります。

## このアップデートで何が変わったか

- AWS FinOps Agent は、FinOps 実践者とエンジニアリングチームを対象とした新しいエージェントサービスであります。
- コスト関連の質問に回答し、最適化の機会を提示し、コスト異常を自動調査し、スケジュール設定したワークフローを実行することができます。
- AWS コストと使用状況データに関するレポート生成が可能であります。
- AWS Cost Optimization Hub と AWS Compute Optimizer からのライトサイジング、アイドルリソース、Savings Plans の推奨事項を提供します。
- Jira チケットの自動作成機能を備えています。
- コスト異常が検出された場合、根本原因を自動調査し、Slack チャネルに結果を投稿することができます。
- 現在、US East (N. Virginia) リージョンで利用可能であり、AWS GovCloud と AWS China リージョンを除くすべての AWS リージョンのコストデータをカバーしています。
- プレビュー期間中は追加料金なしで提供されます。

# Claude Fable 5が一般提供開始〜Mythosレベル性能の安全なAIモデル

AWS announces Claude Fable 5, the first generally available Mythos-class model

**カテゴリ:** What's New
**公開日:** 2026-06-09
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/claude-fable-5-aws/)

このページでは、AWS What's Newで発表された「AWS announces Claude Fable 5, the first generally available Mythos-class model」の内容を日本語で要約し、スライド形式で確認できます。

## 要約

Claude Fable 5は、AWSで一般提供される新しいMythos級のAIモデルであり、強力なセーフガードを備えており、金融、法務、エンジニアリングなどの専門分野において、長時間にわたって自律的に複雑なタスクを実行できます。このアップデートは、エンタープライズ企業や開発者が本番環境のAIアプリケーションを構築する際に、生産性の大幅な向上と信頼性の確保を実現することができます。

## このアップデートで何が変わったか

- Claude Fable 5は、AWSで一般提供されるMythos級の最初のモデルです
- 強力なセーフガードが組み込まれており、より広範な利用に対応した安全性が確保されています
- ほぼすべてのテストベンチマークで最先端の性能を発揮しており、自律的な知識作業とコーディング能力が大幅に向上しています
- 複雑な知識作業とコーディングタスクに対して、長時間にわたって介入なしで実行できます
- 金融、法務、マーケティング、営業、データ、エンジニアリングなどの専門的なタスクに対応しています
- 自動的にスキルを更新し、独自の評価フレームワークを開発し、納品前に作業を検証します
- Amazon BedrockとClaude Platform on AWSの2つのアクセス方法が提供されます
- Amazon Bedrockはデータをawsインフラストラクチャ内に保持し、Guardrails、Knowledge Bases、地域データレジデンシーなどのAWS管理機能を提供します
- Claude Platform on AWSはAnthropicが運営し、統一されたAWS請求と認証を通じてAnthropicのネイティブプラットフォーム体験を提供します

## 対象ユーザー

- エンタープライズ企業の開発者
- AIアプリケーション構築者
- 自動化されたタスク実行が必要な組織
- 金融、法務、マーケティング、営業、データ、エンジニアリングなどの専門分野の企業

## 利用方法

### Amazon Bedrock経由

- データはAWSインフラストラクチャ内に保持
- Guardrails機能で安全性を強化
- Knowledge Basesでナレッジ統合
- 地域データレジデンシーに対応

### Claude Platform on AWS経由

- Anthropic運営のネイティブプラットフォーム
- 統一されたAWS請求システム
- AWS認証との統合

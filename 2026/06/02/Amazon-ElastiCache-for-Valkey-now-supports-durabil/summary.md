# Amazon ElastiCache for Valkey が永続性をサポート

Amazon ElastiCache for Valkey now supports durability

**カテゴリ:** What's New
**公開日:** 2026-06-02T15:00:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/durability-amazon-elasticache)

このページでは、AWS What's Newで発表された「Amazon ElastiCache for Valkey now supports durability」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

Amazon ElastiCacheがValkey 9.0から同期および非同期の2つの永続性オプションをサポートするようになり、マイクロ秒単位の読み取りレイテンシーを維持しながらデータ損失を防ぐことが可能になりました。このアップデートは、AIエージェントのメモリ管理、決済処理、在庫管理など、データの耐久性が重要なワークロードに最適です。

## このアップデートで何が変わったか

- Amazon ElastiCacheがValkey 9.0からデータ永続性機能をサポートするようになりました。
- マイクロ秒単位の読み取りレイテンシーを維持しながら、データ損失を防ぐことができるようになりました。
- 複数のアベイラビリティゾーン（AZ）にわたってマルチAZトランザクショナルログを使用してデータを永続化します。
- 同期書き込みオプションでは、クライアントに応答する前に少なくとも2つのAZにデータを永続化し、ゼロデータ損失を実現します。
- 非同期書き込みオプションでは、クライアントに応答した後にデータを永続化し、マイクロ秒単位の書き込みレイテンシーを維持します。
- 非同期書き込みの場合、障害発生時に最大10秒間のコミットされていないデータが失われる可能性があります。
- すべてのAWSコマーシャルリージョン、AWSチャイナリージョン、およびAWSガバメントクラウド（US）リージョンで利用可能です。
- このアップデートはAIエージェントの長期メモリ、ワークフロー状態、RAGアプリケーション用ナレッジベース、決済トークン化、リアルタイム在庫管理などの用途に適しています。
- AWS Management Console、AWS SDK、またはAWS CLIを使用して新しいElastiCacheクラスターを作成する際に、優先する永続性オプションを選択できます。

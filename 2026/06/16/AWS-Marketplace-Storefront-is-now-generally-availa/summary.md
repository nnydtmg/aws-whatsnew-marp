# AWS Marketplace Storefrontが一般利用可能に

AWS Marketplace Storefront is now generally available

**カテゴリ:** What's New
**公開日:** 2026-06-16T04:00:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-marketplace-storefront/)

このページでは、AWS What's Newで発表された「AWS Marketplace Storefront is now generally available」の内容を日本語で要約し、スライド形式で確認できます。

---

## 要約

AWS Marketplace Storefrontは、AWSパートナーがコード不要でブランド化されたストアフロントを構築し、AWS Marketplaceの請求インフラストラクチャを活用して顧客にソリューションを提供できる新機能です。チャネルパートナーと独立系ソフトウェアベンダーが、顧客ディスカバリーと購買プロセスを簡素化し、ビジネス効率を向上させるのに適しています。

## このアップデートで何が変わったか

- このアップデートはAWS Marketplaceが運営するすべてのAWSリージョンで利用可能です。

## 対象ユーザー

- プライベートオファーテンプレート、承認自動化、SalesforceやHubSpotなどのツールとのネイティブCRM連携により、ディール ワークフローを自動化できます。

## 詳細

- AWS Marketplace Storefrontは、AWSパートナーが自社のウェブサイトやアプリケーション上にブランド化されたソリューションカタログを数時間で作成・展開できる機能です。

- コード不要で完全にブランド化されたストアフロントを構成でき、AWS Marketplaceからのリスティングをインポートして同日中にライブにすることができます。

- 取引はAWS Marketplaceの請求インフラストラクチャを通じて処理され、顧客のAWSインボイスに自動的に表示されるため、別個の決済システムを構築・維持する必要がありません。

- プライベートオファーテンプレート、承認自動化、SalesforceやHubSpotなどのツールとのネイティブCRM連携により、ディール ワークフローを自動化できます。

- チャネルパートナーは複数ベンダーのソリューションを再販する場合、各顧客に対してカスタマイズされたカタログを提示し、チャネルビジネスの成長に応じて拡張することができます。

- このアップデートはAWS Marketplaceが運営するすべてのAWSリージョンで利用可能です。

- チャネルパートナーおよび独立系ソフトウェアベンダーにとって有益です。

- 顧客関係の維持強化を支援し、クラウドマーケットプレイスビジネスの管理を簡素化します。

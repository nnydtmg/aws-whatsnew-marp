# AWS Marketplace Storefrontが一般利用可能に

AWS Marketplace Storefront is now generally available

**What's New** | 2026-06-16T04:00:00

---

## 概要

- AWS Marketplace Storefrontは、AWSパートナーがコード不要でブランド化されたストアフロントを構築し、AWS Marketplaceの請求インフラストラクチャを活用して顧客にソリューションを提供できる新機能です。
- チャネルパートナーと独立系ソフトウェアベンダーが、顧客ディスカバリーと購買プロセスを簡素化し、ビジネス効率を向上させるのに適しています。

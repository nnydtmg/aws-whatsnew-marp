## 要約

AWS Backupは強化された変更追跡機能により、S3バックアップのクロスアカウント・クロスリージョンコピー操作を最大8倍高速化し、追加コストなしですべてのAWSリージョンで利用可能になりました。この改善は、数百万のオブジェクトを持つS3バケットを複数のアカウントやリージョン間で管理する組織にとって特に有効です。

## このアップデートで何が変わったか

- 追加コストなしで、AWS BackupがAmazon S3のバックアップクロスアカウント・クロスリージョンコピーをサポートするすべてのAWSリージョンで利用可能です。

- この更新は、複数のリージョンやアカウント間でS3バックアップを管理する組織にとって特に有益です。

## 詳細

- AWS Backupは、数百万のオブジェクトを持つS3バケットに対して、バックアップコピー操作を最大8倍高速化することができるようになりました。

- この高速化は、強化された変更追跡機能により実現され、バックアップコピー間の変更率が低い場合に特に効果的です。

- 従来は宛先アカウントまたはリージョン内のすべてのオブジェクトをスキャンする必要がありましたが、この改善により不要になりました。

- AWS Backupはオブジェクトイベントをリアルタイムで記録することで、より高速なコピー操作と処理時間の短縮を実現しています。

- この機能強化は、すべての新しいS3バックアップのクロスアカウントおよびクロスリージョンコピージョブに自動的に適用されます。

- 追加コストなしで、AWS BackupがAmazon S3のバックアップクロスアカウント・クロスリージョンコピーをサポートするすべてのAWSリージョンで利用可能です。

- この更新は、複数のリージョンやアカウント間でS3バックアップを管理する組織にとって特に有益です。

# AWS Backup が Amazon S3 バックアップコピーのパフォーマンスを強化

AWS Backup enhances Amazon S3 backup copy performance

**発表日: 2026年6月25日**

---

## 概要

### AWS Backup の S3 コピー機能を大幅強化

- **最大8倍の高速化**: 強化された変更追跡機能により、クロスアカウント・クロスリージョンコピー操作を高速化
- **追加コストなし**: すべての AWS リージョンで利用可能
- **対象**: 数百万のオブジェクトを持つ S3 バケット

---

## 前提・背景

### これまでの課題

- **スキャンの負担**: 従来は宛先アカウントまたはリージョン内の**すべてのオブジェクトをスキャン**する必要があった
- **時間がかかる**: 大規模なS3バケットでのコピー操作が時間を要する
- **マルチリージョン対応**: 複数のリージョンやアカウント間での管理が複雑

### 市場背景

- クラウドストレージの規模拡大に伴い、効率的なバックアップ戦略が重要に
- データレジリエンスと災害復旧(DR)への投資が増加
- エンタープライズのマルチアカウント・マルチリージョン戦略が定着

---

## 変更内容・新機能

### 強化された変更追跡機能

- **オブジェクトイベントのリアルタイム記録**: オブジェクトの変更情報を自動記録
- **差分ベースのコピー**: バックアップ間の変更部分のみを効率的に転送
- **スキャン不要**: 宛先側のすべてのオブジェクトスキャンが不要に

### 自動適用

- すべての**新しい S3 バックアップ**のクロスアカウント・クロスリージョンコピージョブに自動適用
- 既存ジョブにも段階的に展開予定

---

## 効果・メリット

### パフォーマンス向上

- **最大8倍の高速化**: バックアップ間の変更率が低い場合に特に効果的
- **処理時間の短縮**: オブジェクトイベントのリアルタイム記録により、コピー操作が迅速に
- **リソース効率化**: 不要なスキャン処理を削減

### コスト最適化

- **追加コストなし**: 機能強化は追加料金なしで提供
- **運用効率化**: バックアップ・リカバリ時間の短縮により、運用コスト削減

### リスク軽減

- **RPO/RTO の改善**: 高速なコピーにより、目標復旧地点(RPO)と目標復旧時間(RTO)の改善
- **ディザスタリカバリーの強化**: マルチリージョン戦略の効率化

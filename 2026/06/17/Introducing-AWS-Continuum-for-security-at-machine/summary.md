# AWS Continuum で機械速度のセキュリティリスク管理を実現

Introducing AWS Continuum for security at machine speed

## 要約

AWS Continuumは、機械速度でセキュリティリスクの発見から修復までのライフサイクル全体を自動化する新しいサービスです。このアップデートは、脆弱性管理プロセスを効率化し、セキュリティチームが戦略的な意思決定に集中できるようにしたい企業に最適です。

## このアップデートで何が変わったか

- Continuum侵入テストおよびContinuumコードスキャン（プレビュー）が利用可能になりました。

- このアップデートは、セキュリティリスク管理プロセスを自動化し、チーム間の調整時間を削減したい組織に適しています。

## 対象ユーザー

- 確認された露出に対して、ユーザーが定義したガードレール内で迅速で可逆的な軽減策を提供します。

## 詳細

- AWS Continuumは、機械速度でセキュリティリスクを発見、優先順位付け、検証、および修復するサービスです。

- 既存のツールからの検出結果を取り込み、環境とビジネスのコンテキストグラフを使用して各リスクを優先順位付けします。

- 隔離されたサンドボックス環境で再現可能な証拠を構築することで、どのリスクが悪用可能であるかを検証します。

- 確認された露出に対して、ユーザーが定義したガードレール内で迅速で可逆的な軽減策を提供します。

- Continuum脅威モデリングは、設計ドキュメントまたはソースコードから自動的にSTRIDE形式の包括的な脅威モデルを生成します。

- Continuum侵入テストおよびContinuumコードスキャン（プレビュー）が利用可能になりました。

- このサービスは、Amazon GuardDutyおよびAWS Security Hubなどの既存のAWSセキュリティサービスと連携して動作します。

- セキュリティチームは、手動トリアージから方向設定と結果承認へシフトすることができます。

- このアップデートは、セキュリティリスク管理プロセスを自動化し、チーム間の調整時間を削減したい組織に適しています。

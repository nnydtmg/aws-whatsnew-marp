# Amazon CloudWatch、ログ分析機能 Log Analytics を発表

Amazon CloudWatch introduces Log Analytics for unified log analysis

**カテゴリ:** What's New
**公開日:** 2026-06-15T18:00:00
**元記事:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-cloudwatch-log-analytics/)

## 要約

Amazon CloudWatchのLog Analyticsは、Logs Insights、Live Tail、Contributor Insightsを統一されたコンソール体験で提供する新機能であり、ログデータの分析、リアルタイム監視、トップコントリビューター特定を効率的に行いたい組織に適しております。本機能は全ての商用AWSリージョンで利用可能で、既存の料金体系が適用されます。

## このアップデートで何が変わったか

- Amazon CloudWatchがLog Analyticsという新機能を導入いたしました。

- パターン認識、保存されたクエリとパラメータ、ファセット機能、自然言語クエリ生成、ビジュアライゼーションなどの既存のLogs Insights機能がすべて利用可能です。

## 対象ユーザー

- このアップデートはログデータの分析と監視を効率化する必要があるユーザーに適しています。

## 詳細

- Amazon CloudWatchがLog Analyticsという新機能を導入いたしました。

- Log AnalyticsはCloudWatch Logs Insights、Live Tail、Contributor Insightsを統一されたコンソール体験で提供するものです。

- 複数のクエリを異なるタブで実行できるようになりました。

- パターン認識、保存されたクエリとパラメータ、ファセット機能、自然言語クエリ生成、ビジュアライゼーションなどの既存のLogs Insights機能がすべて利用可能です。

- Live TailとContributor InsightsもLog Analytics内からアクセスできるようになりました。

- Log Analyticsは全ての商用AWSリージョンで利用可能です。

- 料金体系は既存のLogs Insights、Live Tail、Contributor Insightsと同じです。

- このアップデートはログデータの分析と監視を効率化する必要があるユーザーに適しています。

- リアルタイムログストリーミングとトップコントリビューターの特定を一元的に行いたい組織に有益です。

- CloudWatchコンソールからLog Analyticsを選択して開始できます。

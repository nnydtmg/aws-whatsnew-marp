# Amazon Lightsail expands blueprint selection with a new WordPress blueprint

**カテゴリ:** What's New
**公開日:** 2026-02-27T23:28:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/wordpress-blueprint-lightsail/](https://aws.amazon.com/about-aws/whats-new/2026/02/wordpress-blueprint-lightsail/)

---

## 要約

Amazon Lightsailの新しいWordPressブループリントにより、セキュリティが強化された状態で、わずか数分でクラウド上にWordPressサイトを立ち上げることができるようになりました。このアップデートは、クラウド初心者やWordPressサイトを迅速に構築したい開発者にとって非常に有用です。

## 詳細

### 主な変更点

- Amazon Lightsailが新しいWordPressブループリントを提供開始
- ワンクリックで数分以内にWordPressがプリインストールされたVPSを起動
- Instance Metadata Service Version 2（IMDSv2）がデフォルトで強制

### セットアップと設定

- ガイド付きセットアップウィザードにより以下を自動処理
  - カスタムドメイン接続
  - DNS設定
  - 静的IPアドレス割り当て
  - Let's EncryptによるHTTPS暗号化

- Lightsailコンソール内で全ての設定が完結
- 複雑な手動設定が不要

### 利用可能性と対象ユーザー

- Lightsailが利用可能な全てのAWSリージョンで利用可能
- クラウド初心者から開発者まで適用
- WordPressサイトを迅速に立ち上げたいユーザー向け

### インスタンスバンドル

- OSやストレージ、月間データ転送量が含まれる
- すぐに運用を開始できる構成
- 予測可能な月額料金体系

## ユースケース

### 推奨される利用シーン

1. **小〜中規模企業のコーポレートサイト**: 簡単なセットアップと運用
2. **個人ブログ・メディア**: オウンドメディア構築
3. **WooCommerceによるECサイト**: オンラインストア構築
4. **ポートフォリオサイト**: 個人・フリーランスのポートフォリオ展示

## メリット

### ユーザー側のメリット

- **時間短縮**: セットアップが数分で完了
- **複雑さの軽減**: 手動設定が不要
- **セキュリティ向上**: IMDSv2 が自動適用
- **コスト効率**: 予測可能な固定料金
- **スケーラビリティ**: 必要に応じてスケール可能

### AWS側のメリット

- AWS初心者の入口としての強化
- クラウドでのWordPress普及促進
- 顧客満足度向上

# Amazon Bedrock announces OpenAI-compatible Projects API

**カテゴリ:** What's New
**公開日:** 2026-02-26T23:06:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-bedrock-projects-api-mantle-inference-engine/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-bedrock-projects-api-mantle-inference-engine/)

---

## 要約

Amazon Bedrockは、Mantle推論エンジンを通じてOpenAI互換のProjects APIをサポート開始いたしました。このAPIにより、複数のプロジェクト間でのより良い分離、IAMベースのアクセス制御、およびタグによるコスト可視性が実現でき、追加料金なくご利用いただけます。

## 詳細

- Amazon Bedrockは、Mantle推論エンジンを通じてOpenAI互換のProjects APIをサポートするようになりました。

- Projects APIにより、複数のアプリケーション、環境、またはチームを持つお客様は、個別のプロジェクトを作成して、それぞれの間でより良い分離を実現できます。

- 各プロジェクトに異なるIAMベースのアクセス制御を割り当てることができます。

- コスト可視性の向上のため、各プロジェクトにタグを追加できます。

- Projects APIはOpenAI互換API、Responses API、Chat Completions APIを使用するすべてのお客様がMantle推論エンジンを通じて利用できます。

- Projects APIの使用に追加料金はかかりません。基盤となるモデル推論の消費に対してのみ料金が発生します。

- このアップデートは、複数のアプリケーション、環境、またはチームを管理する組織に適しています。

- IAMベースのアクセス制御とコスト管理の強化を必要とするお客様に有益です。

## 主なメリット

- **プロジェクト分離**: 複数のアプリケーション・環境・チームを独立して管理
- **セキュリティ強化**: IAMベースのきめ細かいアクセス制御
- **コスト最適化**: タグによるプロジェクト単位のコスト追跡
- **スケーラビリティ**: 追加料金なく柔軟に拡張可能
- **相互運用性**: OpenAI互換APIにより既存ツールとの互換性を実現

## 利用対象

- OpenAI互換API、Responses API、Chat Completions APIを利用するすべてのお客様
- Mantle推論エンジン経由での利用
- すべてのAWSリージョンで利用可能（段階的展開）

# Amazon WorkSpaces Applications extends support for 4K resolution

**カテゴリ:** What's New
**公開日:** 2026-02-25
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-workspaces-applications-4K-resolution/](https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-workspaces-applications-4K-resolution/)

---

## 要約

Amazon WorkSpaces Applicationsは、すべてのクライアント接続モードと非高速化インスタンスタイプで最大4K解像度（4096 x 2160）のサポートを拡張いたしました。この更新により、ユーザーは自身のハードウェア投資に合わせて最適なインスタンスタイプを選択でき、高解像度ディスプレイでも一貫性のある高品質なストリーミング体験を追加費用なしでご利用いただけるようになります。

## 詳細

Amazon WorkSpaces Applicationsが4K解像度（4096 x 2160）のサポートを拡張したことが新機能です。従来はグラフィックス高速化インスタンスのクラシックモードのみで高解像度モニターが対応していましたが、現在はすべてのクライアント接続モードと非高速化インスタンスタイプで4K解像度が利用可能になりました。

### 対応範囲

- **すべての接続モード**: ネイティブアプリケーションモード、クラシックアプリケーションモード、デスクトップビューで最大4K解像度が提供
- **マルチモニター対応**: 最大2台のモニター（各4096 x 2160ピクセル）が利用可能
- **ウルトラワイドモニター対応**: 21:9アスペクト比のウルトラワイドモニターなど、高解像度ディスプレイを使用するユーザーに特に価値

### 利用可能性

- 新機能は**追加費用なし**で、Amazon WorkSpaces Applicationsが利用可能なすべてのAWSリージョンで提供
- 利用要件:
  - 2026年2月4日以降にリリースされたWorkSpaces Applicationsエージェント
  - 2026年2月18日以降にリリースされたManaged WorkSpaces Applicationsイメージ更新

### パフォーマンス

- ハードウェアアクセラレーション機能に関係なく、すべてのインスタンスタイプで一貫性のある高品質なストリーミング体験が提供されます

## 主な利点

1. **柔軟なインスタンス選択**: ユーザーのハードウェア投資に合わせて最適なインスタンスタイプを選択可能
2. **コスト効率化**: 非高速化インスタンスで4K対応により、総コストの最適化
3. **生産性向上**: 高解像度ディスプレイでより多くのコンテンツを表示可能
4. **ユーザー体験向上**: 一貫性のある高品質なストリーミング体験
5. **グローバル提供**: すべてのAWSリージョンで即座に利用可能

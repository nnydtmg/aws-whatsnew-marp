# AWS WAF announces AI activity dashboard for visibility into AI bot and agent traffic

**カテゴリ:** What's New
**公開日:** 2026-02-24T06:00:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/aws-waf-ai-activity-dashboard/](https://aws.amazon.com/about-aws/whats-new/2026/02/aws-waf-ai-activity-dashboard/)

---

## 要約

AWS WAFが新しいAI活動ダッシュボードを発表し、650以上のボットとエージェントの検出により、AIトラフィックの一元的な可視化と管理を実現しました。このアップデートはインフラストラクチャコストの増加やアプリケーションパフォーマンスの低下に対応する必要があるお客様にとって特に有益です。

## 詳細

- AWS WAFが新しいAI活動ダッシュボードを発表し、AIボットとエージェントトラフィックの一元的な可視化を提供するようになりました。

- AWS WAF Bot Controlの検出カバレッジが650以上のユニークなボットとエージェントを追跡できるようになり、業界で最も包括的なAIボット検出カタログの一つを提供しています。

- ダッシュボードではAIトラフィックのトレンド可視化、最もアクティブなボットと頻繁にアクセスされるパスの特定、ボットカテゴリと検証ステータス別のリクエスト量分析が可能です。

- AWS WAF Bot Controlルールを使用して、検証済みのAI検索クローラーを許可しながら、未検証のエージェントをレート制限またはブロックするなどのアクションを直接実行できます。

- 検出カタログはAI検索エンジンクローラー、AIデータコレクター、AIアシスタント、大規模言語モデルトレーニングクローラーなど複数のカテゴリをカバーしており、継続的に更新されています。

- フラットレート料金プランのお客様にはダッシュボードがすべての有料プランに含まれており、その他のWAFお客様は追加費用なしで利用できます。

- この新しいダッシュボードと拡張された検出機能はAWS WAFが利用可能なすべてのAWSリージョンで提供されています。

# AWS IAM Policy Autopilot is now available as a Kiro Power

**カテゴリ:** What's New
**公開日:** 2026-02-23T18:49:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/aws-iam-policy-autopilot-kiro-power/](https://aws.amazon.com/about-aws/whats-new/2026/02/aws-iam-policy-autopilot-kiro-power/)

---

## 要約

AWS IAM Policy Autopilot が Kiro power として利用可能になり、開発者は Kiro IDE から1クリックでインストールして、手動設定なしに AWS IAM ポリシーのベースラインを迅速に作成できるようになりました。このアップデートは、AWS アプリケーションの迅速なプロトタイピングとポリシー作成の効率化を求める開発者およびアーキテクトに最適です。

## 詳細

### ツールの背景

- AWS IAM Policy Autopilot は、2025年の re:Invent で発表されたオープンソースの静的コード分析ツール
- 現在、Kiro power として利用可能になり、AI 支援開発環境にシームレスに統合

### 主な改善点

1. **インストール簡素化**
   - Kiro IDE およびウェブインターフェースから1クリックでインストール可能
   - 手動の MCP サーバー設定が不要になりました

2. **ワークフロー統合**
   - ワークフローが簡素化され、AI 支援開発環境にシームレスに統合
   - 開発者がコーディングワークフローを離れることなくポリシーを生成

3. **ポリシー自動生成**
   - 開発者が AWS IAM ポリシーのベースラインを迅速に作成
   - 手動でのポリシー作成の必要性を排除
   - 97% 以上の不要な権限を削減したセキュアなポリシーを生成

### ユースケース

- **AWS アプリケーションのプロトタイピング**: 迅速なベースラインポリシー作成
- **新規 AWS プロジェクトのセットアップ**: 初期段階でのセキュア設定
- **IDE 環境での開発効率向上**: ワークフローから離れずにポリシー生成

### 対象ユーザー

- AWS IAM ポリシーの作成を効率化したい開発者
- AWS アーキテクト
- AI コーディングアシスタント（Kiro、Claude Code、Cursor）のユーザー

### 対応環境

- **IDE**: Kiro IDE、Claude Code、Cursor
- **言語**: Python、TypeScript、Go
- **インストール方法**: Kiro Powers からの1クリック、または uvx/pip による手動セットアップ

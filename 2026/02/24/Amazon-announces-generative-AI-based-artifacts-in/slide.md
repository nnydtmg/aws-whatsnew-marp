# Amazon Q Developer での生成AI ベースアーティファクト
AWS Management Console でリソースとコストデータを可視化

**発表日: 2026年2月23日**

---

## 概要

### 3つの主要な変更点

- **生成AI ベースのアーティファクト機能**: AWS Management Console でリソースとコストデータを自然言語クエリで可視化
- **使いやすさの向上**: Amazon Q のアイコンがナビゲーションバーに移動し、チャットパネルが左側に配置
- **複数の可視化形式対応**: テーブル形式でリソースデータ、チャート形式でコストデータを表示

### 利用可能なスコープ

- **対応リージョン**: Amazon Q Developer が利用可能なすべての AWS リージョン

---

## 前提・背景

### 企業ユーザーが直面する課題

- AWS リソースの現状把握が複雑化
- コスト分析に手作業が必要
- ナビゲーション階層が深い
- テキストベースの情報では直感的な理解が困難

### 市場トレンド

- 生成 AI を活用したデータ可視化需要の急増
- AWS Management Console での操作性向上への期待
- クラウドコスト最適化への関心の高まり
- 開発者・運用チームの効率化への投資増加

---

## 変更内容・新機能

### 主要機能1: アーティファクト機能の一般提供開始

- 生成 AI ベースの UI で、自然言語クエリに対応
- リソースデータとコストデータの双方に対応
- テーブルとチャート形式の自動選択

### 主要機能2: インターフェースの改善

- **ナビゲーション**: Amazon Q アイコンが移動し、アクセス性が向上
- **レイアウト**: チャットパネルが左側に配置される新しい配置
- **フルスクリーンモード**: アーティファクトパネルで集中モード体験を提供

### 主要機能3: プロンプトライブラリの活用

- Prompt Library のサンプルプロンプトで即座に開始可能
- S3 バケット、RDS コストなど一般的なクエリに対応

## 効果・メリット

### 開発者・運用チームへのメリット

- **効率化**: 複雑な CLI コマンドが不要に
- **直感性**: 自然言語で質問するだけで結果を取得
- **時間削減**: データの収集・加工に要する時間を大幅削減
- **意思決定の高速化**: ビジュアルベースの迅速な認識

### エンタープライズ顧客へのメリット

- **コスト最適化**: 無駄なリソースの早期発見
- **可視性の向上**: ダッシュボード型で一目でリソース状態を把握
- **チーム間の連携**: 同じデータに基づいた意思決定が可能

## ユースケース

### ユースケース1: クラウドコスト管理

**シナリオ**: CFO が月別のクラウド支出の傾向を確認
- Amazon Q に「過去 3 ヶ月のコスト推移」と質問
- 結果がチャート形式で表示され、即座に分析可能

### ユースケース2: リソース最適化

**シナリオ**: DevOps チームが低使用率 EC2 を特定
- 「使用率 10% 以下の EC2 インスタンス」を検索
- テーブル形式で該当リソースが抽出され、削除判断が容易

### ユースケース3: オンボーディング

**シナリオ**: 新規チームメンバーが AWS 環境を理解
- Prompt Library のサンプルで簡単に開始
- ビジュアルな情報で学習曲線を短縮

# Amazon announces generative AI-based artifacts in Amazon Q Developer for visualizing resource and cost data

**カテゴリ:** What's New
**公開日:** 2026-02-23T20:05:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/generative-ai-based-Amazon-Q-artifacts/](https://aws.amazon.com/about-aws/whats-new/2026/02/generative-ai-based-Amazon-Q-artifacts/)

---

## 要約

Amazon Q Developerのアーティファクト機能により、AWS Management Console上でリソースとコストデータを生成AIベースで自然言語クエリを通じてテーブルやチャート形式で可視化できるようになりました。この機能は、AWSリソースの管理とコスト分析を効率化したいエンタープライズ顧客および開発・運用チームに特に有用です。

## 詳細

- Amazon Q Developerが、AWS Management Consoleでリソースとコストデータを可視化するための生成AIベースのアーティファクト機能を一般提供開始したことが新機能です。
- テーブル形式でリソースデータを表示し、チャート形式でコストデータを表示することができます。
- Amazon Qのアイコンがナビゲーションバーに移動し、チャットパネルが左側に配置されて、使いやすさが向上しました。
- S3バケットやRDSコストなどのクエリに対して、自然言語で質問することで、テーブルやグラフで結果を可視化できます。
- Prompt Libraryのサンプルプロンプトを利用して、すぐに始めることができます。
- アーティファクトパネルはチャットパネルの隣に表示され、フルスクリーンモードで専用の集中モード体験が可能です。
- この機能は、Amazon Q Developerが利用可能なすべてのAWSリージョンで提供されます。
- この更新は、AWSリソースの状態とコストを効率的に理解・管理したいエンタープライズ顧客や、データ駆動型の意思決定を必要とするチーム向けに最適です。
- AWS Management Consoleでリソース管理を行う開発者やオペレーション担当者にとって、生成AIを活用した直感的なデータ可視化ツールとして有用です。

# Amazon Aurora DSQL now integrates with Kiro powers and AI agent skills

**カテゴリ:** What's New  
**公開日:** 2026年2月18日  
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-aurora-dsql-integrates-with-kiro-powers-and-agent-skills](https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-aurora-dsql-integrates-with-kiro-powers-and-agent-skills)

---

## 要約

Amazon Aurora DSQLがKiro powersおよびAIエージェントスキルと統合され、開発者はAIアシスト開発によってAurora DSQLアプリケーションをより効率的に構築できるようになりました。このアップデートは、スキーマ設計、パフォーマンス最適化、データベース操作に関する専門知識を複数のAIエージェントに提供し、開発サイクルを短縮します。

## 詳細

### 主な変更点

1. **Kiro Powers との統合**
   - Kiro powersレジストリを通じて、開発者はKiro IDE内でワンクリックインストールでAurora DSQL用のキュレーションされたMCPサーバーにアクセス可能
   - 複雑な設定が不要で、すぐに利用開始できる

2. **Aurora DSQL Model Context Protocol（MCP）サーバー**
   - 開発のベストプラクティスとともにバンドルされている
   - AIエージェントが以下の領域でサポートを提供:
     - スキーマ設計
     - パフォーマンス最適化
     - データベース操作

3. **複数のAIコーディングエージェント対応**
   - Aurora DSQLスキルはSkills CLIを介して以下エージェントに対応:
     - Kiro CLI
     - Claude Code
     - Gemini
     - Cursor
     - Copilot
     - Cline
     - Windsurf
     - Roo
     - OpenCode
     - Codex

4. **簡素化されたインストール**
   - 開発者はシングルコマンドでスキルをインストール可能
   - 複雑な設定ファイル編集が不要

5. **動的ガイダンス**
   - PostgreSQL互換SQLパターン
   - 分散データベース設計
   - IAM認証
   - これらのガイダンスは動的に提供される

6. **継続的な更新**
   - Aurora DSQLが新機能を追加する際、スキルリリースには更新されたパターンとガイダンスが含まれる
   - エージェントが常に最新のベストプラクティスにアクセス可能

## 効果・メリット

### 開発効率の向上
- AI エージェントによる自動サポートで開発時間を削減
- 開発サイクルが短縮される
- ベストプラクティスが自動で提供される

### スキルギャップの埋め合わせ
- 経験が浅い開発者も質の高いコードを作成可能
- データベース設計の専門知識がなくても最適化されたスキーマが設計できる

### 品質向上
- AI がパターンマッチングにより一般的なミスを検出・修正
- ベストプラクティスに基づいた設計で本番環境のトラブルを防止

### 柔軟性
- 複数のAIエージェント対応により、開発者が好きなツールを選択可能
- 継続的な更新により最新の技術トレンドに対応

## ユースケース

1. **新規プロジェクト**: スキーマ設計段階から AI エージェントがベストプラクティスを提案
2. **既存システム最適化**: パフォーマンスボトルネック分析とチューニング
3. **マイグレーション**: 従来型DB → Aurora DSQL への移行支援
4. **チームスキル向上**: 経験差のあるチーム全体の開発品質を向上

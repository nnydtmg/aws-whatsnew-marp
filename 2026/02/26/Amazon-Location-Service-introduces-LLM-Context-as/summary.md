# Amazon Location Service introduces LLM Context as a Kiro power and Claude Code plugin to improve AI performance

**カテゴリ:** What's New
**公開日:** 2026-02-25T16:26:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-location-service-introduces-kiro-power-claude-skill-llm-context/](https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-location-service-introduces-kiro-power-claude-skill-llm-context/)

---

## 要約

Amazon Location Serviceは、LLMコンテキストをKiro power、Claude Codeプラグイン、およびオープンAgent Skills形式として導入し、開発者がロケーション対応機能の実装を加速できるようになりました。このアップデートは、AIを活用して開発効率を向上させたい開発者およびソリューションアーキテクトに最適です。

## 詳細

**新機能について:**
- Amazon Location Serviceが、Kiro power、Claude Codeプラグイン、およびオープンAgent Skills形式のエージェントスキルとしてLLMコンテキストを導入しました。

- キュレーションされたAIエージェントコンテキストにより、開発者はKiro、Claude Code、Cursorなどの生成AIツールを使用してコード精度を向上させることができます。

- 提供されるコンテキストには、配送アプリケーションのアドレス入力フォーム、地図表示、最寄り店舗検索、ルート可視化などの一般的なロケーションベースのソリューション向けの事前検証済み実装パターンと段階的な手順が含まれています。

- 開発者はAPIインテグレーションの詳細ではなく、アプリケーション固有のロジックに集中することができるようになります。

**このアップデートが適している対象者:**
- Amazon Location Serviceを使用してロケーション対応機能をアプリケーションに追加する開発者。

- AIおよびLLMを活用して開発を加速させたいソリューションアーキテクト。

- 複数のAIエージェント（Kiro、Claude Code、Cursorなど）を使用している開発チーム。

- ジオスペーシャルデータと位置情報機能の実装を効率化したい企業。

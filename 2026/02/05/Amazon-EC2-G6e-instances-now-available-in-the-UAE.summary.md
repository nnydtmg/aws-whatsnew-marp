# Amazon EC2 G6e instances now available in the UAE region

**カテゴリ:** What's New
**公開日:** 2026-02-05
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-ec2-g6e-instances-uae-region/](https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-ec2-g6e-instances-uae-region/)

---

## 要約

Amazon EC2 G6e インスタンスが中東（UAE）リージョンで新たに利用可能になり、NVIDIA L40S GPU を搭載した高性能なコンピューティング環境を提供いたします。本更新は、大規模言語モデルのデプロイ、画像生成、3D シミュレーション、デジタルツイン構築などの AI および空間コンピューティングワークロードを実行する必要がある組織に特に適しております。

## 詳細

- Amazon EC2 G6e インスタンスが、2026年2月5日より中東（UAE）リージョンで利用可能になりました。
- G6e インスタンスは、NVIDIA L40S Tensor Core GPU により駆動されており、機械学習と空間コンピューティングの幅広いユースケースに対応しています。
- 大規模言語モデル（LLM）や画像、動画、音声生成用の拡散モデルのデプロイに使用できます。
- 3D シミュレーションとデジタルツインの作成により、空間コンピューティングワークロードの能力を拡張できます。
- 最大8個の NVIDIA L40S Tensor Core GPU（GPU あたり 48 GB メモリ）、第3世代 AMD EPYC プロセッサを搭載しています。
- 最大 192 vCPU、最大 400 Gbps のネットワーク帯域幅、最大 1.536 TB のシステムメモリ、最大 7.6 TB のローカル NVMe SSD ストレージをサポートしています。
- 本更新は、機械学習エンジニア、AI 研究者、空間コンピューティング開発者、および大規模モデルのデプロイが必要な組織に適しています。
- オンデマンドインスタンス、予約インスタンス、スポットインスタンス、またはセービングスプランとして購入できます。
- 現在、米国、アジア太平洋、中東、ヨーロッパの複数リージョンで利用可能です。

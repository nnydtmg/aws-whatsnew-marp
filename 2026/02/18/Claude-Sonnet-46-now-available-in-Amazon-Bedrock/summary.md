# Claude Sonnet 4.6 now available in Amazon Bedrock

**カテゴリ:** What's New
**公開日:** 2026-02-17T15:43:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/claude-sonnet-4.6-available-in-amazon-bedrock/](https://aws.amazon.com/about-aws/whats-new/2026/02/claude-sonnet-4.6-available-in-amazon-bedrock/)

---

## 要約

Claude Sonnet 4.6はAmazon Bedrockで利用可能となり、コーディングからエンタープライズアプリケーションまで幅広い用途において、Opus 4.6に近いインテリジェンスをより低いコストで提供します。開発者とエンタープライズチームは、高品質なマルチステップオーケストレーション、エージェンティックワークフロー、ドメイン固有のアプリケーション開発に活用でき、Sonnet 4.5からの移行も容易です。

## 詳細

- Claude Sonnet 4.6がAmazon Bedrockで利用可能になりました。
- このモデルはコーディング、エージェント、プロフェッショナルワークにおいて最先端のパフォーマンスを提供します。
- ブラウザベースの自動化をほぼ人間レベルの信頼性で展開できます。
- Claude Opus 4.6に近いインテリジェンスをより低いコストで実現します。
- Sonnet 4.5からの直接的なアップグレードであり、会話品質と効率的なマルチステップオーケストレーションが向上しています。
- 検索とチャットアプリケーションにおいて、シングルターンおよびマルチターン交換で信頼性の高いパフォーマンスを提供します。
- 開発者はエージェンティックワークフロー、リードエージェント、サブエージェントの役割を活用できます。
- エンタープライズチームはスプレッドシート作成、財務モデル作成、コンプライアンスレビュー、データ要約タスクに活用できます。
- Sonnet 4.5からの移行には最小限のプロンプト調整のみが必要です。

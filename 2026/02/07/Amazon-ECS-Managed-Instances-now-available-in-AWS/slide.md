# Amazon ECS Managed Instances now available in AWS European Sovereign Cloud

**What's New** | 2026-02-06T17:10:00

---

## 概要

- Amazon ECS Managed InstancesがAWS European Sovereign Cloudで利用可能になり、インフラストラクチャ管理の負担を軽減しながら自動スケーリングとセキュリティ強化を実現するサービスです。
- このアップデートは、インフラ運用の複雑さを排除し、コスト最適化を求めるエンタープライズ組織やコンテナワークロード運用チームに特に有益です。

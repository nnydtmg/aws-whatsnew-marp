# AWS Builder ID now supports Sign in with Apple

**カテゴリ:** What's New
**公開日:** 2026-02-05T19:20:00
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/02/aws-builder-id-sign-in-apple](https://aws.amazon.com/about-aws/whats-new/2026/02/aws-builder-id-sign-in-apple)

---

## 要約

AWS Builder ID が Apple Sign in に対応しました。Google Sign in に続いて、Apple ユーザーが AWS Builder Center、AWS Training and Certification、AWS re:Post、AWS Startups、および Kiro へのアクセスを Apple Account 認証情報でシームレスに実行できるようになりました。

## 詳細

AWS Builder ID に Sign in with Apple が追加されました。これにより、パスワード管理の複雑さが排除され、忘れたパスワードの問題が軽減され、新規ユーザー登録と既存ユーザーのサインインの両方に対してシームレスな体験が提供されます。開発者やビルダーは、AWS Builder Center での開発リソース、認定プログラム登録、AWS re:Post でのコミュニティディスカッション、スタートアップリソース探索、または Kiro での開発に Apple Account を安全なゲートウェイとして使用できます。

# Claude Opus 4.6 now available in Amazon Bedrock

**カテゴリ:** AI/ML Services
**公開日:** 2026-02-05
**元記事:** [https://aws.amazon.com/about-aws/whats-new/2026/2/claude-opus-4.6-available-amazon-bedrock/](https://aws.amazon.com/about-aws/whats-new/2026/2/claude-opus-4.6-available-amazon-bedrock/)

## 要約

Amazon Bedrock now supports Claude Opus 4.6, Anthropic's most intelligent model. The model excels at agentic tasks, complex coding projects, and enterprise workflows with deep reasoning and multi-step orchestration capabilities. It supports 200K and 1M token context windows for processing extensive documents and codebases.

## 詳細

Claude Opus 4.6 is now available on Amazon Bedrock starting February 5, 2026. Key capabilities include:

### 主要機能

**Agentic Workflows**
- Manages complex tasks across dozens of tools with industry-leading reliability
- Proactively spinning up subagents with minimal human oversight
- Multi-step task orchestration with autonomous coordination

**Coding Excellence**
- Handles long-horizon projects and complex implementations
- Works with large-scale codebases from requirements gathering through maintenance
- Supports multi-million-line codebase migrations
- Full lifecycle development support

**Enterprise Workflows**
- Financial analysis: transforms multi-day manual work into automated insights
- Cybersecurity: detects subtle attack patterns
- Computer use: automates data movement between applications
- Professional-grade workflow quality

**Extended Context**
- 200K tokens (standard) for document processing
- 1M tokens (preview) for massive codebases and comprehensive document analysis
- Enables processing of 750,000+ words or 3,000+ pages simultaneously

**Multi-step Reasoning**
- Sophisticated reasoning for complex problem-solving
- Deep orchestration of multi-step tasks
- Self-correction capabilities

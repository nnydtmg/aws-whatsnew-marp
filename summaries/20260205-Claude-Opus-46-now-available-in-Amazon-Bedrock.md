## 要約

Amazon Bedrock now supports Claude Opus 4.6, Anthropic's most intelligent model designed for coding, enterprise agents, and professional work. It features industry-leading performance for agentic tasks, complex coding projects, and enterprise workflows with deep reasoning capabilities, 200K and 1M context tokens support.

## 詳細

Claude Opus 4.6 brings advanced capabilities to Amazon Bedrock customers with:
- Industry-leading performance for agentic tasks managing complex workflows across dozens of tools with reliability and autonomous subagent coordination
- Advanced coding capabilities for long-horizon projects, complex implementations, and large-scale codebases with full lifecycle support
- Enterprise-grade workflows including financial analysis, cybersecurity applications, and computer use workflows
- Support for 200K context tokens (standard) and 1M context tokens (preview)
- Available across multiple AWS regions

# AWS Builder ID now supports Sign in with Apple

**What's New** | 2026-02-05

---

## 概要

- AWS Builder IDがAppleでのサインイン機能に対応し、開発者はAppleアカウント認証情報を使用してAWSアプリケーションにアクセスできるようになりました。
- このアップデートにより、パスワード管理の複雑性が軽減され、新規登録と既存ユーザーの両方にとってシームレスなサインイン体験が実現されます。

---

## 前提・背景

### これまでの課題

AWS Builder IDがAppleでのサインイン機能に対応しました。Googleでのサインインに加えて、Appleアカウントを使用した認証が可能になりました。AWS Builder Center、AWS Training and Certification、AWS re:Post、AWS Startups、およびKiroなどのAWSアプリケーションへのアクセスが簡素化されます。パスワード管理の複雑性が軽減され、パスワード忘れの問題が減少します。新規ユーザー登録と既存ユーザーのサインインの両方でシームレスな体験が提供されます。

## 変更内容・新機能

### Sign in with Appleの統合

AWS Builder IDがAppleでのサインイン機能に対応し、開発者はAppleアカウント認証情報を使用してAWSアプリケーションにアクセスできるようになりました。このアップデートにより、パスワード管理の複雑性が軽減され、新規登録と既存ユーザーの両方にとってシームレスなサインイン体験が実現されます。

- **登録の簡素化**: 新規ユーザーにとってシームレスな登録体験
- **セキュリティ**: Apple認証による安全な認証方式
- **アクセスの効率化**: AWSリソースへのアクセスが安全で効率的に

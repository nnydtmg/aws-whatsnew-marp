# Claude Opus 4.6 now available in Amazon Bedrock

**AI/ML** | 2026-02-05

---

## 概要

- Amazon Bedrock now supports Claude Opus 4.6
- Anthropic's most intelligent model for coding, enterprise agents, and professional work
- Industry-leading performance for agentic tasks and complex coding projects
- Advanced enterprise workflows with deep reasoning capabilities
- Support for 200K and 1M context tokens

## 変更内容・新機能

### Claude Opus 4.6 の主要機能

**Agentic Tasks対応**
- 複数ツール（dozens）の統合管理
- 自動的なサブエージェント起動
- 高い信頼性と自律的な調整
- 人間の監視削減

**Coding Capabilities**
- 長期のコーディングプロジェクト対応
- 複雑な実装処理
- 大規模コードベース処理
- ライフサイクル全体をサポート（要件定義→実装→保守）

**Enterprise Workflows**
- 金融分析（数日の手作業を自動化）
- サイバーセキュリティ（微細な攻撃パターン検出）
- Computer Use Workflows（アプリ間のデータ移動）

**Context Token Support**
- 200K トークン（標準）
- 1M トークン（プレビュー）
- 大規模ドキュメント・コードベース対応
